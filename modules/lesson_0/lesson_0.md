## 🔤 Греческий алфавит и произношение

| Буква       | Звук | Пример (Город/Страна)      | Название |
|:------------|:-----|:---------------------------|:---------|
| **Α, α**    | [a]  | **Αλβανία** (Албания)      | άλφα     |
| **Β, β**    | [v]  | **Βολιβία** (Боливия)      | βήτα     |
| **Γ, γ**    | [γ]  | **Γαλλία** (Франция)       | γάμα     |
| **Δ, δ**    | [δ]  | **Δανία** (Дания)          | δέλτα    |
| **Ε, ε**    | [e]  | **Ελλάδα** (Греция)        | έψιλον   |
| **Ζ, ζ**    | [z]  | **Ζαΐρ** (Заир)            | ζήτα     |
| **Η, η**    | [i]  | **ΗΠΑ** (США)              | ήτα      |
| **Θ, θ**    | [θ]  | **Θιβέτ** (Тибет)          | θήτα     |
| **Ι, ι**    | [i]  | **Ιράν** (Иран)            | γιώτα    |
| **Κ, κ**    | [k]  | **Καναδάς** (Канада)       | κάπα     |
| **Λ, λ**    | [l]  | **Λίβανος** (Ливан)        | λάμδα    |
| **Μ, μ**    | [m]  | **Μαρόκο** (Марокко)       | μι       |
| **Ν, ν**    | [n]  | **Νότια Αφρική** (ЮАР)     | νι       |
| **Ξ, ξ**    | [ks] | **Μεξικό** (Мексика)       | ξι       |
| **Ο, ο**    | [o]  | **Ολλανδία** (Голландия)   | όμικρον  |
| **Π, π**    | [p]  | **Πακιστάν** (Пакистан)    | πι       |
| **Ρ, ρ**    | [r]  | **Ρωσία** (Россия)         | ρο       |
| **Σ, σ, ς** | [s]  | **Σερβία** (Сербия)        | σίγμα    |
| **Τ, τ**    | [t]  | **Τουρκία** (Турция)       | ταυ      |
| **Υ, υ**    | [i]  | **Υεμένη** (Йемен)         | ύψιλον   |
| **Φ, φ**    | [f]  | **Φιλιππίνες** (Филиппины) | φι       |
| **Χ, χ**    | [h]  | **Χιλή** (Чили)            | χι       |
| **Ψ, ψ**    | [ps] | **Ψ** (Пси)                | ψι       |
| **Ω, ω**    | [o]  | **Πολωνία** (Польша)       | ωμέγα    |
| **ΟΥ, ου**  | [u]  | **Ουκρανία** (Украина)     | -        |

> **Правила на заметку:**
> * Буква **ς** пишется только на конце слов (напр. *Καναδάς*), в начале и середине — **σ** (*Σερβία*).
> * Звук **[i]** могут передавать буквы: **η, ι, υ**.
> * Звук **[o]** могут передавать буквы: **ο, ω**.
> * Сочетание **ου** всегда читается как **[у]**.
