## 📋 Таблица 1. Тип A — спряжение глаголов μένω и κάνω (настоящее время)

| Русское местоимение | Ελληνικά | μένω      | κάνω      | Окончание |
|---------------------|----------|-----------|-----------|-----------|
| Я                   | Εγώ      | μένω      | κάνω      | -ω        |
| Ты                  | Εσύ      | μένεις    | κάνεις    | -εις      |
| Он                  | Αυτός    | μένει     | κάνει     | -ει       |
| Она                 | Αυτή     | μένει     | κάνει     | -ει       |
| Оно                 | Αυτό     | μένει     | κάνει     | -ει       |
| Мы                  | Εμείς    | μένουμε   | κάνουμε   | -ουμε     |
| Вы                  | Εσείς    | μένετε    | κάνετε    | -ετε      |
| Они (м.р.)          | Αυτοί    | μένουν(ε) | κάνουν(ε) | -ουν(ε)   |
| Они (ж.р.)          | Αυτές    | μένουν(ε) | κάνουν(ε) | -ουν(ε)   |
| Они (ср.р.)         | Αυτά     | μένουν(ε) | κάνουν(ε) | -ουν(ε)   |

**Пояснение:**
Глаголы типа A имеют одинаковые окончания в настоящем времени. Форма *(ε)* в скобках — разговорный вариант.

## 📋 Таблица 2. Примеры предложений с глаголом μένω

| Πρόταση                                 | Перевод                        |
|-----------------------------------------|--------------------------------|
| Εγώ μένω στο Γουδί.                     | Я живу в Гуди.                 |
| Εσύ πού μένεις;                         | Где ты живёшь?                 |
| Ο Παναγιώτης μένει στον Πειραιά.        | Панайотис живёт в Пирее.       |
| Η Αρλέτα μένει Μουσών 23.               | Арлета живёт на ул. Мусон 23.  |
| Το σκυλάκι μένει στο σπίτι.             | Собака живёт дома.             |
| Εμείς μένουμε στην Καισαριανή.          | Мы живём в Кесариани.          |
| Εσείς μένετε εδώ;                       | Вы живёте здесь?               |
| Ο Φου και ο Παναγιώτης δε μένουν κοντά. | Фу и Панайотис живут не рядом. |
| Η Αρλέτα και η Άννα μένουν στη Φιλοθέη. | Арлета и Анна живут в Филофее. |
| Τα παιδιά μένουν στην Κίνα τώρα.        | Дети сейчас живут в Китае.     |

## 📍 Таблица 3. Роды в иминительном падеже

| Род     | Артикль | Типичные окончания | Примеры   |
|---------|---------|--------------------|-----------|
| Мужской | ο       | -ος                | ο Γιώργος |
|         |         | -ας                | ο Κώστας  |
|         |         | -ης                | ο Γιάννης |
| Женский | η       | -α                 | η Μαρία   |
|         |         | -η                 | η Ελένη   |
| Средний | το      | -ο                 | το τρένο  |
|         |         | -ι                 | το παιδί  |
|         |         | -μα                | το γράμμα |

**Пояснение:**

- Род в греческом определяется артиклем: ο / η / το.
- Окончания — подсказка, но всегда учи слово с артиклем.
