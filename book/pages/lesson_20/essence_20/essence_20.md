## 🧠 Грамматика

### Условные конструкции (**αν**)

**Ан** + **подъярусное** (или форма с **να**): во второй части — **будущее** (**θα** + …) **или** **повелительное** (совет, приказ в момент речи).

| Образец | Перевод |
| :--- | :--- |
| Αν βρέξει, θα μαζέψεις τα ρούχα από το μπαλκόνι. | Если пойдёт дождь — уберёшь бельё с балкона. |
| Αν βρέξει, μάζεψε τα ρούχα από το μπαλκόνι. | Если пойдёт дождь — убери бельё (повелит.). |

### Тип **Α** (смена тематического гласного в аористе подъярусного)

| Группа | Настоящее | Υποτακτική | Προστακτική (εσύ) | Προστακτική (εσείς) |
| :--- | :--- | :--- | :--- | :--- |
| **-νω, -ζω → σ** | αφήνω | να αφήσω | άφησε | αφήστε |
| | διαβάζω | **να διαβάσω** | διάβασε | διαβάστε |
| **-ζω, -γω, -χω, -χνω → ξ** | κοιτάζω | να κοιτάξω | κοίταξε | κοιτάξτε |
| | ανοίγω | να ανοίξω | άνοιξε | ανοίξτε |
| | τρέχω | να τρέξω | τρέξε | τρέξτε |
| | διώχνω | να διώξω | δίωξε | διώξτε |
| **-εύω, -φω, -βω → ψ** | ταξιδεύω | να ταξιδέψω | ταξίδεψε | ταξιδέψτε |
| | βάφω | να βάψω | βάψε | βάψτε |
| | κρύβω | να κρύψω | κρύψε | κρύψτε |

*Примечание:* в одной печатной таблице для **διαβάζω** в столбце подъярусного иногда встречается **να αγοράσω** — это **не** форма от **διαβάζω**; норма: **να διαβάσω**.

### Женские на **-η** (**-ση, -ξη, -ψη**)

**Единственное число:** **η** … **-η** · **της** … **-ης** · **τη(ν)** … **-η**.

**Множественное:** **οι** … **-εις** · **των** … **-εων** · **τις** … **-εις**.

Примеры из урока:

- Αυτή η **άσκηση** είναι εύκολη. Κοίταξες την **ημερομηνία λήξης** στο γάλα; Πρέπει να κάνω μια **επίσκεψη** στον οφθαλμίατρο.
- Υπάρχει πολλή **ανεργία** και λίγες **θέσεις εργασίας**. Οι **δάσκαλοι** των μικρών **τάξεων** θα μιλήσουν αύριο με τους **γονείς**. Πείτε μας τις **σκέψεις** σας γι’ αυτό το **θέμα**.

Сводка (модель **η στάση**):

| Падеж | ед. | мн. |
| :--- | :--- | :--- |
| именительный | η στάση | οι στάσεις |
| родительный | της στάσης | των στάσεων |
| винительный | τη στάση | τις στάσεις |

## 📋 Эталоны упражнений

### 1 — предложения с **αν** (образец + эталоны)

**Образец:** Βρέχει → μαζεύεις τα ρούχα από το μπαλκόνι  
→ *Αν βρέξει, θα μαζέψεις τα ρούχα από το μπαλκόνι.*  
→ *Αν βρέξει, μάζεψε τα ρούχα από το μπαλκόνι.*

| № | Пара | Эталон (θα …) | Эталон (προστακτική) |
| :---: | :--- | :--- | :--- |
| 2 | Μαγειρεύεις → καλείς τον Γιάννη για φαγητό | Αν μαγειρέψεις, θα καλέσεις τον Γιάννη για φαγητό. | Αν μαγειρέψεις, κάλεσε τον Γιάννη για φαγητό. |
| 3 | Περνάς από τον φούρνο → παίρνεις ένα μπουκάλι γάλα | Αν περάσεις από τον φούρνο, θα πάρεις ένα μπουκάλι γάλα. | Αν περάσεις από τον φούρνο, πάρε ένα μπουκάλι γάλα. |
| 4 | Έχετε πυρετό → πηγαίνετε στον γιατρό | Αν έχετε πυρετό, θα πάτε στον γιατρό. | Αν έχετε πυρετό, πηγαίνετε στον γιατρό. |
| 5 | Τελειώνεις τις δουλειές νωρίς → μου τηλεφωνείς | Αν τελειώσεις τις δουλειές νωρίς, θα μου τηλεφωνήσεις. | Αν τελειώσεις τις δουλειές νωρίς, τηλεφώνησέ μου. |
| 6 | Το παιδί πεινάει → ετοιμάζεις λίγο γάλα | Αν πεινάει το παιδί, θα ετοιμάσεις λίγο γάλα. | Αν πεινάει το παιδί, ετοίμασε λίγο γάλα. |

### 2 — диалог **Κατερίνα / Μάριος** (номера из πίνακας)

**Πίνακας:** 1 … 5 — как в учебнике.

**Διάλογος (эталон):**

**Κατερίνα:** Τι είναι αυτό;  
**Μάριος:** Εισιτήριο για τον Βόλο. Φεύγω σε δύο ώρες.  
**Κατερίνα:** **1** (Μα γιατί δεν πάς με το αυτοκίνητο; Θα είναι πιο γρήγορα.)  
**Μάριος:** Ναι, αλλά θα είναι πιο κουραστικά.  
**Κατερίνα:** Γιατί το λες;  
**Μάριος:** Γιατί δε μ’ αρέσει καθόλου η οδήγηση. **3** (Προτιμώ να διαβάσω το βιβλίο μου στο τρένο.)  
**Κατερίνα:** Δεν έχεις άδικο. Η διαδρομή δεν είναι άσχημη και, τέλος πάντων, θα πάς πιο άνετα και πιο χαλαρά. **2** (Μην ξεχάσεις να πάρεις κάτι για τον δρόμο. Το ταξίδι ανοίγει την όρεξη.)  
**Μάριος:** Έφαγα καλό πρωινό. Δεν πεινάω καθόλου.  
**Κατερίνα:** **4** (Δεν πεινάς τώρα;), αλλά θα πεινάσεις πιο μετά.

### 4 — записка **Άννα** → **Άγγελος**

**Α** *(παίρνω, βγάζω, βάζω, κλείνω)* — эталон:

Άγγελε, Όλα σχεδόν έτοιμα για το ταξίδι μας! Πήρα τηλέφωνο και **έκλεισα** τα εισιτήρια. **Έβγαλα** τα πράγματά μου, **έβαλα** και μερικά δικά σου ρούχα στο μεγάλο σακίδιο. **Έκλεισα** στο μπαλκόνι τα καναρίνια. Θα τα προσέχει η κυρία Αρετή.

**Β** *(πηγαίνω, αργώ, γυρίζω, πίνω, κάνω, συναντώ)* — эталон:

Το απόγευμα θα πάω να πάρω τα εισιτήρια από το ταξιδιωτικό γραφείο και μετά **θα συναντήσω** την Ειρήνη. **Θα κάνω** μία βόλτα και **θα πιώ** ένα καφεδάκι στην πλατεία. Δε **θα αργήσω**. **Θα γυρίσω** στο σπίτι κατά τις 10:00 το βράδυ.

**Γ** *(έρχομαι, ποτίζω, φτιάχνω, παρκάρω, δίνω)* — эталон:

… πρέπει να έρθεις με το αυτοκίνητο και **να παρκάρεις** κάπου κοντά. Μην ξεχάσεις **να ποτίσεις** τα λουλούδια και **να δώσεις** τα κλειδιά στην κυρία Αρετή. Μήπως μπορείς **να φτιάξεις** και τα τοστ για το ταξίδι;

## 🎧 Задания на прослушивание (лексика и замена без аудио)

В учебнике к блокам **«Έχετε όλα τα δικαιολογητικά;»** и **«Για ιατρικές εξετάσεις…»** прилагается **аудиозапись**. Без неё нельзя выполнить задание «отметь ✓» или «расставь 7 шагов» **точно как в классе**. Ниже — **замкнутая** опора: список документов с переводом и **модельный мини-диалог** только из этой лексики (для чтения вслух).

### Документы (таблица к первому заданию)

| :--- | :--- |

| Υπεύθυνη δήλωση | справка-заявление (официальная) |

**Заголовок задания (перевод):** «У вас все подтверждающие документы?»

**Инструкция (сжатый перевод):** Прослушать **дважды** диалог между сотрудником **ΙΚΑ** и женщиной, которая хочет оформить **βιβλιάριο υγείας** (медицинский полис); во время прослушивания поставить **✓** под **ΝΑΙ**, если документ **нужен**, и под **ΟΧΙ**, если **не нужен**.

### Мини-диалог (чтение вслух; лексика из списка)

**Υπάλληλος:** Καλημέρα σας. Θέλετε βιβλιάριο υγείας;  
**Κυρία:** Ναι, παρακαλώ.  
**Υπάλληλος:** Έχετε υπεύθυνη δήλωση και πιστοποιητικό γέννησης;  
**Κυρία:** Ναι, τα έχω.  
**Υπάλληλος:** Και ιατρικές εξετάσεις;  
**Κυρία:** Όχι ακόμα.

| Реплика | Перевод |
| :--- | :--- |
| Καλημέρα σας. | Доброе утро. |
| Θέλετε βιβλιάριο υγείας; | Хотите оформить медполис? |
| Ναι, παρακαλώ. | Да, пожалуйста. |
| Έχετε …; | У вас есть …? |

| Όχι ακόμα. | Пока нет. |

### Второе задание (Людмила, медосмотры)

**Заголовок:** «Для медицинских обследований…»

**Инструкция (сжатый перевод):** Прослушать **дважды**, как женщина рассказывает знакомому, **куда ходила** и **какие обследования** делала для **άδεια διαμονής** (вида на жительство); затем **расставить по порядку** **семь** пунктов (без примера).

**Вопрос:** *Τι έκανε η Λουντμίλα την Τρίτη το πρωί;* — Что делала Людмила во вторник утром?

Без аудио можно **тренировать произношение**: читать строки таблицы и переводить по смыслу; порядок шагов проверяет только запись в учебнике.
