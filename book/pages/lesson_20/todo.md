## Контент и учебная ценность (в приоритете)

### `essence_20/essence_20.md`

**Статус:** файла конспекта **нет** — для читателя нет самодостаточного слоя «грамматика → фразы → [эталоны] → словарь» ([essence-generation.md](../../../docs/essence-generation.md)).

**Что важно вынести из главы для изучения и отработки (ориентир по `lesson_20_digitized.md`)**

- **Ядро грамматики:** сводные таблицы настоящего, подъярусного наклонения и повелительного (включая типы A и B в терминологии главы, «аномальные» группы, глаголы на -βω/-γω/-νω, блок с окончаниями повелительного -ες / -είτε), плюс краткое русскоязычное правило и **минимальные пары** для запоминания окончаний — без отсылок к цветным зонам учебника как к обязательному источнику.
- **Склонение женских на -η (-ση, -ξη, -ψη):** примеры предложений из главы + компактная парадигма (единственное / множественное) и лексический стол «по окончаниям» — это большой объём лексики урока; в конспекте он должен быть **внутри файла**.
- **Коммуникативные опоры:** условные конструкции (αν + будущее / повелительное) по образцу упр. 1; **полный** диалог упр. 2 с подставленными номерами (эталон в файле, без пропусков); ключевые реплики упр. 3–4 (повелительное, просьбы).
- **Темы «отдых / работа / услуги»:** сжатые эталоны или готовые фразы по блокам про отели, объявления о работе (соответствие профессия — номер), монологические вопросы и ролевые сценарии из финала главы (заголовки совпадают с оцифровкой) — чтобы повторение не требовало файлов сырья.
- **Задания на аудирование (в учебнике: B32, B33):** без аудиотрека читатель не может выполнить задачу «как в книге»; в конспекте имеет смысл либо **кратко обозначить** это ограничение и дать **замкнутую** альтернативу (например, лексика документов + модельный мини-диалог на греческом с переводом для чтения вслух), либо иной согласованный с репозиторием приём — главное: не оставлять смысл только в формулировке «прослушайте дважды».
- **Итоговая таблица лексики** (последний блок раздела со словарём по стандарту репозитория): все **уникальные** слова и формы из материала урока на греческом с русским переводом, **включая заголовки и условия заданий** (требование [agents.md](../../../AGENTS.md) / essence-generation).

### `lesson_voice_20/voice_lesson_20.md`

**Что уже полезно**

- Охвачены **разные пласты** главы: условные конструкции и диалог в поезде; заголовок и список документов к заданию про ИКА; сценарий с перестановкой шагов (B33); отели и часть объявлений о работе; монологические вопросы и ролевые сценарии из заключительных разделов главы.
- В **Блоке 3** названы три грамматических термина урока и даны тематические группы (документы, отдых, глаголы).

**Пробелы и риски для отработки**

1. **Шапка по стандарту репозитория:** в [voice-lesson-from-digitized.md](../../../docs/voice-lesson-from-digitized.md) и [agents.md](../../../AGENTS.md) для учебных `.md` ожидаются **хлебные крошки** и таблица **«Быстрые ссылки»** (как минимум на `content_20.md`). Сейчас файл сразу начинается с блоков — хуже навигация и расхождение с чеклистом после генерации.
2. **Несоответствие промпту [voice_lesson_generation.md](../../../docs/promts/voice_lesson_generation.md):** в ситуациях **1**, **2**, **3** у **Роль А / Роль Б** нет **пошаговых** инструкций на русском (часто сведено к «прочитайте образец» / «расставьте по смыслу») — для ученика это слабее, чем в других уроках (ср. формулировки ролей в `voice_lesson_11.md`).
3. **Перевод в чек-листах:** длинные **греческие** формулировки условий **B32** и **B33** в чек-листах **без дословного или сжатого русского перевода** — конфликт с требованием промпта: *«Ни одно греческое слово или фраза в словаре и шпаргалках не должны оставаться без точного перевода»* (для чек-листов это тоже относится к «шпаргалкам» по смыслу задания).
4. **Блок 2 (шпаргалка)** очень **узкий** по сравнению с объёмом главы: почти нет опор по **повелительному** и **подъярусному** для характерных глаголов, фраз для **бронирования**, **собеседования**, типовых **вопросов в гостинице** — при том что это центральная коммуникативная тема урока.
5. **Блок 3 (словарь):** не отражает богатую **лексику на -ση / -ξη / -ψη**, названия **профессий** из объявлений, топонимы и детали объявлений; для урока с упором на формы глагола и существительные читатель получает **недостаточную** опору для самостоятельной тренировки по тексту главы.
6. **Ситуация 4:** в чек-лист перечислены не все **восемь** первых объявлений (есть 1, 2, 3, 7 — нет 4, 5, 6, 8); при желании охватить вакансии — **симметрично** добавить недостающие строки с переводом или явно сузить заголовок ситуации.
7. **Аудирование:** без записи диалогов ситуации **2–3** остаются в основном **чтением списков**; для голосовой отработки полезнее добавить **короткие модельные реплики** (по смыслу из главы / комментария в оцифровке), не выдумывая новую лексику вне `lesson_20_digitized.md`.

## Итог

Главный вывод: **конспект урока 20 отсутствует** — без него материал не закрывает стандарт «повторение без внешних источников». **Голосовой урок** есть, но **слабее по объёму шпаргалки и словаря**, местами **не дотягивает до шаблона ролей и перевода** из промпта; имеет смысл **доработать** после появления **essence** или параллельно с ним, не раздувая выдуманной лексикой вне оцифровки.
