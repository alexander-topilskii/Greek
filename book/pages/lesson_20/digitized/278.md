**20** *ενότητα*

---

**Πάμε πάλι!** *(τίτλος · μπορντό, πάνω στο βιβλίο.)*

---

Σύνοψη κλίσης: **ενεστώτας**, **υποτακτική**, **προστακτική** (πρόσωπα **εσύ** / **εσείς**). Οι πίνακες στο βιβλίο είναι χρωματικά διαχωρισμένα (μπλε, κίτρινο / πορτοκαλί, πράσινο, μωβ).

---

### Τύπος Α *(μπλε περιοχή)*

| Αλλαγή θέματος | Ενεστώτας | ΥΠΟΤΑΚΤΙΚΗ | ΠΡΟΣΤΑΚΤΙΚΗ (εσύ) | ΠΡΟΣΤΑΚΤΙΚΗ (εσείς) |
| :--- | :--- | :--- | :--- | :--- |
| **-νω, -ζω → σ** | αφήνω | να αφήσω | άφησε | αφήστε |
| | διαβάζω | να αγοράσω | διάβασε | διαβάστε |
| **-ζω, -γω, -χω, -χνω → ξ** | κοιτάζω | να κοιτάξω | κοίταξε | κοιτάξτε |
| | ανοίγω | να ανοίξω | άνοιξε | ανοίξτε |
| | τρέχω | να τρέξω | τρέξε | τρέξτε |
| | διώχνω | να διώξω | δίωξε | διώξτε |
| **-εύω, -φω, -βω → ψ** | ταξιδεύω | να ταξιδέψω | ταξίδεψε | ταξιδέψτε |
| | βάφω | να βάψω | βάψε | βάψτε |
| | κρύβω | να κρύψω | κρύψε | κρύψτε |

*Στη γραμμή «διαβάζω» το βιβλίο δίνει «να αγοράσω» — πιθανό τυπογραφικό (αναμενόμενο: «να διαβάσω»).*

### Ρήματα σε **-βω**, **-γω**, **-θω**, **-νω** *(πράσινο)*

| Ενεστώτας | ΥΠΟΤΑΚΤΙΚΗ (-βω, -γω, -θω, -νω) | ΠΡΟΣΤΑΚΤΙΚΗ (εσύ) | ΠΡΟΣΤΑΚΤΙΚΗ (εσείς) -τε |
| :--- | :--- | :--- | :--- |
| γίνομαι | να γίνω | γίνε | γίνετε |
| κάνω | να κάνω | κάνε | κάνετε (κάντε) |
| καταλαβαίνω | να καταλάβω | κατάλαβε | καταλάβετε |
| μαθαίνω | να μάθω | μάθε | μάθετε |
| μένω | να μείνω | μείνε | μείνετε |
| περιμένω | να περιμένω | περίμενε | περιμένετε (περιμέντε) |
| πηγαίνω | να πάω | πήγαινε | πηγαίνετε (πηγαίντε) |
| πλένω | να πλύνω | πλύνε | πλύνετε (πλύντε) |
| φεύγω | να φύγω | φύγε | φύγετε |
| **έρχομαι** | να έρθω | έλα | ελάτε |
| **κάθομαι** | να καθίσω / κάτσω | κάθισε / κάτσω | καθίστε |

### Ανώμαλες λήξεις προστακτικής **-ες**, **-είτε** *(μωβ)*

| Ενεστώτας | ΥΠΟΤΑΚΤΙΚΗ | ΠΡΟΣΤΑΚΤΙΚΗ (εσύ) -ες | ΠΡΟΣΤΑΚΤΙΚΗ (εσείς) -είτε |
