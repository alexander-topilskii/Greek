### Содержание и методика

- **Итоговая таблица в конце словаря**: по правилам — лексика этого урока (стр. 59–65); сейчас явно заявлено объединение `essence_1…essence_5`. Для повторения объём оправдан, но формально не совпадает с чеклистом в essence-generation; таблица перегружена строками вроде ссылок на «стр. 14», коды A26–A34 — это мета других уроков, а не компактный словарь главы 5. Имеет смысл сузить таблицу до материала 59–65 с явной пометкой для читателя или вынести кумулятивный список в отдельный суммарный файл (если так решите архитектурно).
- **После итоговой таблицы** идёт блок «Опора для продуктивного задания» и снова ссылка на `digitized/65.md`. По п. 2 essence-generation итоговая таблица должна быть последним блоком файла (чеклист «могу ли я…» — выше таблицы). Сейчас порядок нарушен.
- **Неполные реплики в блоке готовых фраз**: в диалоге Фоивос / Фу / Ли у Фу средняя реплика сведена к «…», хотя в оцифровке есть полный текст (про «πώς λένε… ταξιδεύεις»). Для отработки по памяти нужна полная версия без многоточия-подмены.
- **Пробелы в эталонах** относительно оцифрованной главы:
  - **Упр. 7 (оба блока «Διαλέγω», стр. 62)** — в конспекте нет явного ключа (в т.ч. варианты про Петра, Марианну, телефон); в словаре есть пометка про аудио, но самодостаточная подсказка для самопроверки без звука отсутствует. Либо дать ключ, либо чётко описать альтернативу внутри конспекта, без отсылки к файлам.
  - **Упр. 12 (акростих, ответы по строкам для слова «испанка»)** и **Упр. 13 (языки по странам)** — в конспекте не видно полного набора эталонов; для закрепления лексики стран и национальностей полезно добавить готовые ответы, как для соседних упражнений.
- **Вводный абзац** с названием курса «Ελληνικά Α'» — по смыслу близко к привязке к изданию; для строгой автономности лучше нейтральная формулировка («обзор уроков 0–4» без бренда пособия).

## `lesson_voice_5/voice_lesson_5.md`

### Оформление (voice-lesson-from-digitized)

- **Нет хлебных крошек и таблицы «Быстрые ссылки»** с `content_5.md` сразу после `#` — требование документации; сейчас файл начинается с `### Блок 1`.
- **Подзаголовок** «(по оцифровке главы)» для читателя избыточен и намекает на внешний источник; лучше нейтральное название в духе других материалов для ученика.

### Качество материала для отработки

- **Мусор из пайплайна / мета-отсылки**: в блоке про притяжательность в шпаргалке вставлен фрагмент про «χειρόγραφο»; в чек-листах — «как в оцифровке», «как в синих фразах». Убрать или заменить нейтральными пояснениями (без отсылки к цвету блока учебника и к файлу оцифровки).
- **Греческая строка в чек-листе ситуации 1** (про проживание в Пирее и Кипсели): первая часть обрывается на точке перед «в Кипсели» без повторения глагола; лучше три полных коротких предложения или одна строка через слэши, как в учебном блоке главы.
- **Пропуски в чек-листе ситуации 4** (`________`) снижают ценность для самостоятельной проговорки: либо дать готовые формы (*είμαι* / *είναι* / *Είμαστε*), либо вынести в отдельный подпункт «допишите сами».
- **Тематический словарь**: для урока с большим объёмом топонимов и стран (в т.ч. фонетика 1.7–1.9) текущие таблицы покрывают ядро, но не отражают всю тренировочную лексику стр. 65 — при желании расширить группу топонимов и стран без выхода за пределы `lesson_5_digitized.md`.
