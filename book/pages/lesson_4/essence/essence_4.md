# 📋 Конспект: урок 4 (Μιλάς ελληνικά;) — A1

*Урок 4.*

## 1. Глагол **μιλάω** / **μιλώ** («говорить» — о языке)

- Две параллельные формы первого лица: **μιλάω** и **μιλώ**.

| Лицо             | Форма                    |
|:-----------------|:-------------------------|
| εγώ              | μιλάω / μιλώ             |
| εσύ              | μιλάς                    |
| αυτός — αυτή — αυτό | μιλάει / μιλά         |
| εμείς            | μιλάμε                   |
| εσείς            | μιλάτε                   |
| αυτοί / αυτές / αυτά | μιλάνε / μιλούν     |

- **Μιλάς ελληνικά;** — Ты говоришь по-гречески?  
- **Μιλάτε αγγλικά;** — Вы говорите по-английски?

---

## 2. Глаголы: знаю / вижу / ищу / понимаю

| Глагол        | Значение (в контексте урока)     |
|:--------------|:---------------------------------|
| **ξέρω**      | знать (человека, объект, факт)   |
| **βλέπω**     | видеть                           |
| **ψάχνω**     | искать                           |
| **καταλαβαίνω** | понимать (речь, смысл)        |

- После них с дополнением-объекта обычно **артикль в винительном падеже**.

---

## 3. Винительный падеж: **τον / την / το**

| Род     | Артикль |
|:--------|:--------|
| мужской | τον     |
| женский | την     |
| средний | το      |

### Примеры

- *Ξέρω **τον** Πέτρο.* · *Βλέπεις **την** τράπεζα;* · *Ψάχνω **το** πάρκο.* / **τον κινηματογράφο.**

- В диалогах урока: **Ξέρετε το πάρκο Ελευθερίας;** · **Ψάχνω τον κινηματογράφο…** · **Βλέπετε την Εθνική Τράπεζα;**

---

## 4. Национальность: **Είμαι …**

- Формы по роду (**-ος** / **-α** или **-η** по учебному списку).

| Значение (примеры из учебника) | Муж.      | Жен.       |
|:-------------------------------|:----------|:-----------|
| грек                            | Έλληνας   | Ελληνίδα   |
| испанец / испанка               | Ισπανός   | Ισπανίδα   |
| русский / русская               | Ρώσος     | Ρωσίδα     |
| австралиец …                    | Αυστραλός | Αυστραλέζα |

- В тексте также встречаются: **Αλβανός / Αλβανή**, **Ιρακινός / Ιρακινή**, **Κινέζος / Κινέζα** и др. — ориентир: таблица **Είμαι … (υπηκοότητα)** в оцифровке урока.

---

## 5. Языки: средний род, **без** артикля

| Язык        | По-гречески |
|:------------|:------------|
| греческий   | ελληνικά    |
| английский  | αγγλικά     |
| испанский   | ισπανικά    |
| русский     | ρωσικά      |
| итальянский | ιταλικά     |
| немецкий    | γερμανικά   |
| французский | γαλλικά     |
| арабский    | αραβικά     |

- **Миλάω ελληνικά.** · **Δεν μιλάω ισπανικά.** · **Ο Μάρκος μιλάει αγγλικά, ιταλικά και αραβικά.**

---

## 6. Степень владения (наречия после **μιλάω** или в ответах)

| Греческий         | Перевод          |
|:------------------|:-----------------|
| πολύ καλά         | очень хорошо     |
| αρκετά καλά       | довольно хорошо  |
| καλά              | хорошо           |
| λίγο              | немного          |
| όχι και τόσο καλά | не очень хорошо  |
| καθόλου           | совсем нет       |

- *Μιλάω ρωσικά **πολύ καλά**, γαλλικά **αρκετά καλά**.*  
- Ответ без полного предложения: **Λίγο.** / **Όχι, καθόλου.**

---

## 7. Отрицание: **δεν** + глагол

- *Δεν **μιλάω** ελληνικά.*  
- *Δεν **καταλαβαίνω.** / Δεν καταλαβαίνω ελληνικά.*  
- Удвоение объектного местоимения: *Οχι, **δεν τον ξέρω**.* (ответ на *Ξέρεις τον Πέτρο;* — по образцу урока)

---

## 8. Частые вопросы и короткие ответы

### Языки

- **Μιλάς ελληνικά;** — *Ναι, μιλάω.* / *Όχι, δεν μιλάω.*  
- **Μιλάτε ισπανικά;** — *Λίγο.* / *Όχι, καθόλου.*

### Люди / места (винительный)

- **Ξέρεις τον Πέτρο;** — *Ναι, τον ξέρω.* / *Δεν τον ξέρω.*  
- **Βλέπεις την τράπεζα;** — *Ναι, τη βλέπω.*

---

## 9. Притяжательные суффиксы (**μου**, **σου** …)

- Стоят **после** существительного: **το κινητό μου**, **η μητέρα μου**, **ο φίλος σου.**

|      | Перевод (зависит от рода слова слева) |
|:-----|:---------------------------------------|
| μου  | мой / моя / моё …                      |
| σου  | твой …                                 |
| του/της | его / её                          |
| μας/σας | наш / ваш                     |
| τους | их                                     |

---

## 10. **αλλά** — «но»

- *Καταλαβαίνω ελληνικά, **αλλά** δεν μιλάω τόσο καλά.*  
- **Ο Λαφάου καταλαβαίνει ελληνικά, αλλά δε μιλάει τόσο καλά.**

---

## 11. Фразы ситуации «на улице» / «не понимаю»

- **Συγνώμη, δεν καταλαβαίνω…** — Извините, я не понимаю…  
- **Δεν μιλάτε ελληνικά;** / **Αγγλικά;** — элементы перевода между языками.  
- **Ψάχνω τον κινηματογράφο…** — Ищу кинотеатр …  
- **Ευχαριστώ.** — Спасибо.

Дополнительно:

- **εδώ** — здесь · **εκεί** — там.  
- **Ναι, εδώ είναι.** — типичная подсказка о месте.

---

## 12. Числа **100–1000** (кратко)

- **εκατό** (100) → перед единицами в составе составного числа учебник использует **εκατόν**: *εκατόν ένα*, *εκατόν δέκα*.  
- Сотни: **διακόσια**, **τριακόσια**, **τετρακόσια**, **πεντακόσια**, **εξακόσια**, **επτακόσια**, **οκτακόσια**, **εννιακόσια**.  
- **χίλια** — тысяча.
