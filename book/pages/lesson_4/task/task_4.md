# 📝 Практика говорения: урок 4 (Μιλάς ελληνικά;)

*Урок 4.*

## Ситуация 1: Знакомство — откуда и какие языки

**Описание:** Короткая беседа: имя, национальность, на каких языках говорите и с каким уровнем.

**Ход диалога:**

1. **Με λένε …** или **Είμαι ο/η …**
2. **Από πού είσαι;** (**Είσαι …;**) → **Είμαι Έλληνας / Ελληνίδα**, **Ιταλός** … (форма по роду).
3. **Μιλάς ελληνικά;** / **Τι γλώσσα μιλάς;**
4. Ответ со степенью: **μιλάω … πολύ καλά**, **αρκετά καλά**, **λίγο …**, или **δεν μιλάω καθόλου …**
5. B задаёт те же блоки к A.

**Подсказки:**

- Выбор языка без артикля: **ελληνικά**, **αγγλικά**, **ρωσικά** …
- При необходимости вежливо: **Τι γλώσσα μιλάτε;** / **Μιλάτε γερμανικά;**

---

## Ситуация 2: «Не понимаю» и просьба перевести / говорить на другом языке

**Описание:** Сбой понимания; просьба о другой языке; краткая оценка «немного» / «совсем нет».

**Ход диалога:**

1. **Συγνώμη, δεν καταλαβαίνω.** (при желании добавить **… ελληνικά.**)
2. **Μιλάτε ισπανικά;** (**Αγγλικά;**) — партнёр: **Λίγο.** / **Όχι, καθόλου.** / **Ναι, μιλάω.**
3. Уточнение: понимание лучше, чем активная речь — **Καταλαβαίνω ελληνικά, αλλά δεν μιλάω τόσο καλά.**

**Подсказки:**

- **Ούτε εγώ.** / **Ούτε και εγώ δε μιλάω …** можно подставить по смыслу второго упражнений урока.
- Сокращённая частица перед глаголом в разговоре: **δε μιλάω** (как в учебном тексте).

---

## Ситуация 3: Знаешь ли человека (**ξέρω** + **τον / την / το**)

**Описание:** Узнаёте ли вы кого-то из знакомых.

**Ход диалога:**

1. **Ξέρεις τον Πέτρο;** / **Ξέρεις την Μαρία;** (выберите имя или по роли «преподаватель», «однокурсник»).
2. *Ναι, **τον** ξέρω.* / *Όχι, **δεν τον** ξέρω.*
3. Обратный тур вопросов к партнёру.

**Подсказки:**

- Согласование **τον / την / το** с родом упомянутого человека или названия места («знаете ли музей», **ξέρετε το μουσείο** — по желанию).

---

## Ситуация 4: В городе — ищу / вижу ориентир

**Описание:** Ориентируетесь на улице: ищете место (**ψάχνω**), видите ориентир (**βλέπω**).

**Ход диалога:**

1. **Ξέρετε το …;** — отработайте **ψάχνω**: **Ψάχνω τον κινηματογράφο** … / **ψάχνω το μετρό** …
2. **Βλέπετε την τράπεζα;** — *Ναι, **τη** βλέπω.*
3. **Είναι εδώ.** / **Εκεί.** — финал с **Ευχαριστώ.**

**Подсказки:**

- Связка с уроком 2–3 для топонима: если нужно **στην Οδό …**, повторите **Μένω / Είναι στην …**.
- Из учебника: **Ναι, εδώ είναι.** — здесь (это здесь).

---

## Ситуация 5: Портрет человека (языки + понимание + работа или статус по желанию)

**Описание:** По образцу профилей урока: национальность, языки, при желании профессия.

**Ход диалога:**

1. **Ποια είναι αυτή;** / **Ποιος είναι αυτός;** → имя или «это наш преподаватель» (**Είναι η …**, **Ο …**).
2. **Τι γλώσσα μιλάει;** / **Μιλάει ελληνικά πολύ καλά ή λίγο;**
3. **Είναι Ελληνίδα;** — ответ **Ναι / Όχι** с национальностью (**Όχι. Είναι Βουλγάρα.** — по образцу урока).
4. По желанию: **Τι δουλειά κάνει;** → **Είναι γιατρός.** / другое из лексики урока 3.

**Подсказки:**

- Связаные конструкции: **μιλάει βουλγαρικά, αγγλικά και λίγο ελληνικά.**
