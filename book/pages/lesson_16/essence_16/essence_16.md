## 📌 Тема и цели

- Строить **цепочки с **να**** после **θέλω, πρέπει, μπορώ, προτιμώ, λέω, σκέφτομαι, προσπαθώ, έχω, πάω** и др., с правильной **формой глагола** (часто совпадает с **θα + …**).
- Понимать **смену темы** у глаголов на **–νω / –ζω / –γω …** → **–σω / –ξω / –ψω** в **будущем** и после **να**.
- Употреблять **δε θα …** и **να μην …** для отрицания планов.
- Отличать смысл **έχω να + глагол** («нужно сделать / давно не…») от **πρέπει να**.
- Вести разговор о **фильмах, театре, музыке, туре и отеле** на опорных фразах урока.

## 🧠 Грамматика и правила

### Одна парадигма — **θα** и **να**

*После **να** стоит форма **настоящего субъективного** (в учебной традиции — «сослагательное»): в большинстве случаев она **совпадает по основе** с **θα + будущее**.*

**Канонические ряды — смена темы**

**–σω** · **–νω**, **–ζω** → **–σω**

| Ενεστώτας | θα + μέλλ. | να + υποτ. |
|-----------|------------|------------|
| πληρώνω | θα πληρώσω | να πληρώσω |
| αγοράζω | θα αγοράσω | να αγοράσω |

**–ξω** · **–ζω**, **–γω**, **–χω**, **–χνω** → **–ξω**

| Ενεστώτας | θα + μέλλ. | να + υποτ. |
|-----------|------------|------------|
| αλλάζω | θα αλλάξω | να αλλάξω |
| ανοίγω | θα ανοίξω | να ανοίξω |
| προσέχω | θα προσέξω | να προσέξω |
| φτιάχνω | θα φτιάξω | να φτιάξω |

**–ψω** · **–εύω**, **–πω**, **–φω**, **–βω** → **ψω**

| Ενεστώτας | θα + μέλλ. | να + υποτ. |
|-----------|------------|------------|
| δουλεύω | θα δουλέψω | να δουλέψω |
| λείπω | θα λείψω | να λείψω |
| γράφω | θα γράψω | να γράψω |
| κλέβω | θα κλέψω | να κλέψω |

### Частые глаголы **без** смены темы

| Ενεστώτας | θα + μέλλ. | να + υποτ. |
|-----------|------------|------------|
| είμαι | θα είμαι | να είμαι |

### Тип **Β1 + Β2** (фрагмент учебника)

| Ενεστώτας | μέλλοντας | να + |
|-----------|------------|------|
| μιλάω (-ώ) | θα μιλήσω | να μιλήσω |
| τηλεφωνώ | θα τηλεφωνήσω | να τηλεφωνήσω |
| διψάω (-ώ) | θα διψάσω | να διψάσω |
| πονάω (-ώ) | θα πονέσω | να πονέσω |

### Отрицание

| Φράση | Σημείωση |
|-------|----------|
| Δε θα πάω σινεμά. | отрицание **будущего** |
| Λέω να μην πάω σινεμά. | отрицание после **να** — **να μην** |

### Коммуникативные **цепочки с **να****

*Опорные модели (спряжение второго глагола — по лицам, как в блоках **θέλω να…** и **πρέπει να…** в материале урока).*

- **Μπορώ να διαβάσω.** **Θέλω να διαβάσω σήμερα.** **Προτιμώ να διαβάσω στο γραφείο.** **Λέω να διαβάσω…** **Σκέφτομαι να…** **Προσπαθώ να…** **Έχω να διαβάσω για το τεστ.** **Πρέπει να διαβάσω…** **Μπορεί να διαβάσω αργότερα…** **Πάω να διαβάσω στη βιβλιοθήκη.**

*Дополнительно в главе:* **Αποφασίζω να πάω στην Κίνα.** **Χρειάζεται να βγάλω βίζα.** **Καλό είναι να κλείσω εισιτήρια νωρίς.**

### **Έχω να…**

- **Έχω να ανανεώσω το διαβατήριό μου.** *(≈ нужно / пора обновить паспорт.)*
- **Έχω να πάω διακοπές δύο χρόνια!** *(два года «не было» отпуска — букв. «имею накопившееся» пойти.)*

## 💬 Готовые фразы и диалоги

### Рамка «Πάμε βόλτα;»

- Πάμε να πιούμε κανέναν καφέ;
- Θέλετε να δούμε καμιά ταινία;
- Αποφάσισα να κάνω ένα μεγάλο ταξίδι.

### B18 — στο διάλειμμα · στην τάξη

**Ελένη:** Ουφ! Πονοκέφαλος μ’ έπιασε. Σήμερα δεν έχω κέφι για μάθημα. Πάμε να πιούμε κανέναν καφέ στην πλατεία;

**Πάμπλο:** Δηλαδή, λες να φύγουμε; Τι θα πει ο δάσκαλος;

**Ελένη:** Ο Νίκος; Μπορούμε να του πούμε την αλήθεια: ότι είμαστε κουρασμένοι και προτιμάμε να πάμε βόλτα.

**Πάμπλο:** Καλή ιδέα. Να σου πω... Γιατί δεν το λέμε σε όλους; Μπορεί να θέλουν να έρθουν μαζί μας.

**Πάμπλο:** Παιδιά, η Ελένη προτείνει να κάνουμε το μάθημα έξω σήμερα! Στο καφενείο της πλατείας. Τι λέτε; Νίκο, πάμε;

**Δήμητρα-Βόιτσεκ-Χαμίντ:** Ναι, ναι!

**Νίκος:** Ένα λεπτό. Ξέρετε ότι έχουμε πολλή δουλειά. Δεν είπαμε να κάνουμε επανάληψη στα ρήματα σήμερα; Και πρέπει να τελειώσουμε το βιβλίο πριν από τις εξετάσεις. Μπορούμε να βγούμε μετά το μάθημα. Σύμφωνοι;

**Δήμητρα:** Εγώ δεν μπορώ να έρθω μετά. Έχω να γράψω μια εργασία και να στείλω ένα επείγον e-mail.

**Βόιτσεκ:** Κι εγώ πρέπει να ξυπνήσω πολύ νωρίς αύριο.

**Χαμίντ:** Νομίζω ότι δεν έγινε και τίποτα, αν χάσουμε μια ώρα...

**Νίκος:** Λοιπόν, το συζητάμε σε λίγο. Θέλω πρώτα να δούμε τον διάλογο στο σχετικό τμήμα. Αρχίζουμε;

### B19 — στο καφενείο

**Νίκος:** Τώρα που καταλαβαίνετε καλά τα ελληνικά, σκέφτομαι να δούμε καμιά ταινία στο μάθημα.

**Ελένη:** Τέλεια! Δεν ξέρω τίποτα από ελληνικό σινεμά.

**Φού:** Μπορούμε να κοιτάξουμε τι παίζουν οι κινηματογράφοι και να πάμε σε μια ελληνική ταινία.

**Πάμπλο:** Γιατί δεν κανονίζουμε να πάμε και στο θέατρο καμιά φορά;

**Λουντμίλα:** Μπράβο, Πάμπλο! Λατρεύω το θέατρο. Κι έχει μια πολύ καλή παράσταση στο Εθνικό. Αλλά πρέπει να κλείσουμε εισιτήρια.

**Χαμίντ:** Θέλετε να αφήσουμε το θέατρο για αργότερα; Θα είναι δύσκολο να καταλάβουμε, δε νομίζετε; Προτείνω να πάμε κάπου να ακούσουμε ελληνική μουσική.

**Δήμητρα:** Ναι, ναι! Μ’ αρέσουν πολύ τα μπουζούκια! Εκεί μπορείς να χορέψεις και να διασκεδάσεις όλη νύχτα!

**Χαμίντ:** Ποια μπουζούκια, βρε Δήμητρα; Εγώ είπα να πάμε σε καμιά συναυλία με έντεχνη μουσική. Χατζιδάκις, Θεοδωράκης…

**Βόιτσεκ:** Γιατί δεν πάμε καλύτερα σε κανένα ταβερνάκι να φάμε, να πιούμε και να ακούσουμε ρεμπέτικα;

**Νίκος:** Έχω μια ιδέα. Μπορείτε να ψάξετε στις εφημερίδες τι γίνεται στην πόλη αυτή την εβδομάδα και την άλλη φορά αποφασίζουμε τι θέλουμε να κάνουμε.

**Όλοι μαζί:** Εντάξει. Έγινε.

### Лексика досуга (зелёная рамка — опора)

**Κινηματογράφος:** θερινός κινηματογράφος, προβολή, ώρες προβολής, κωμωδία, περιπέτεια, ιστορική ταινία, δραματική ταινία, κοινωνική ταινία, ταινία τρόμου – θρίλερ, κινούμενα σχέδια.

**Μουσική:** κλασική, τζαζ, ροκ, ποπ, παραδοσιακή, λαϊκή, έντεχνη…, συναυλία.

**Θέατρο:** παράσταση (απογευματινή, βραδινή), κάνω κράτηση / κλείνω εισιτήρια, ταβέρνα με ζωντανή μουσική, κέντρο διασκέδασης, μπουζούκια, εισιτήριο (κανονικό, μειωμένο).

**Ευχές:** Καλή διασκέδαση! Καλά να περάσετε!

### Кратко о жанрах (стр. 218–219)

**Ρεμπέτικο** — городская «народная» музыка конца XIX — начала XX в.; влияние беженцев из Малой Азии (1922); темы: любовь, маргинализация, тюрьма; инструменты: **μπουζούκι**, **μπαγλαμάς**, **κιθάρα** и др.; имена: **Μάρκος Βαμβακάρης**, **Βασίλης Τσιτσάνης**, **Ρόζα Εσκενάζυ** …

**Λαϊκό** — с 1950-х, популяризация через радио и кино; **«βαρύ λαϊκό»** (социальные темы), **«ελαφρολαϊκό»**; исполнители: **Στέλιος Καζαντζίδης**, **Μανώλης Αγγελόπουλος** …

**Έντεχνο** — с начала 1960-х, композиторы **Μίκης Θεοδωράκης**, **Μάνος Χατζιδάκης**; европейские черты, связь с поэзией; политический отклик в годы диктатуры (1967–1974); концерты на открытом воздухе.

### B20 — στο καφενείο

**Νίκος:** Λοιπόν, παιδιά, θέλω να σας ζητήσω κάτι.

**Φου:** Ευχαρίστως. Τι;

**Νίκος:** Μπορείτε να μου κάνετε δυο τρία μαθήματα κινέζικων;

**Λι:** Εεε; Τι λες τώρα;

**Νίκος:** Όχι πολλά πράγματα… Τα βασικά μόνο.

**Φου:** Πλάκα μας κάνεις…

**Νίκος:** Όχι βρε, τι πλάκα. Αποφάσισα να πάω στην Κίνα το καλοκαίρι. Έχω δύο χρόνια να πάω διακοπές και θέλω να κάνω ένα μεγάλο ταξίδι.

**Λι:** Έλα! Μη μου πεις!

**Νίκος:** Γι’ αυτό καλό είναι να ξέρω πώς λένε «καλημέρα», «δεν καταλαβαίνω κινέζικα»…

**Παναγιώτης:** Δηλαδή, είσαι έτοιμος; Έβγαλες εισιτήρια;

**Νίκος:** Όχι ακόμα. Αλλά ρώτησα σε δύο τουριστικά γραφεία. Να, εδώ είναι και τα φυλλάδια με τις προσφορές. Περιλαμβάνουν αεροπορικά εισιτήρια, διαμονή σε μονόκλινο σε ξενοδοχείο τεσσάρων αστέρων και όλες τις μετακινήσεις μέσα στη χώρα. Α, και ξεναγό, βέβαια.

**Φου:** Για να δω… Λέω να διαλέξεις το ΚΟΣΜΟΣ. Είναι πιο φτηνό.

**Λι:** Ναι, αλλά το ΠΑΝΟΡΑΜΑ προσφέρει ημιδιατροφή, ενώ το άλλο μόνο πρωινό.

**Νίκος:** Έχει δίκιο η Λι. Και η διαφορά στην τιμή δεν είναι μεγάλη.

**Λι:** Ξέρεις, χρειάζεται να βγάλεις βίζα.

**Νίκος:** Ναι, βέβαια. Θα πάω στην πρεσβεία, αλλά πρέπει πρώτα να ανανεώσω το διαβατήριό μου. Λοιπόν, παιδιά, πότε ξεκινάμε τα μαθήματα;

### «Πώς το λένε;» (после B20)

**Κίτρινο:** Πλάκα κάνεις. · Έλα! Μη μου πεις!

**Πράσινο:** Χρειάζεται να ανανεώσω το διαβατήριο και να βγάλω βίζα.

**Μπλε — Περιλαμβάνει:** αεροπορικά εισιτήρια, διαμονή σε μονόκλινο, ημιδιατροφή, μετακινήσεις, ξεναγό.

### Ταξίδια / Τουρισμός (двустолбцовый блок)

*Левая колонка:* ο επιβάτης, η εκδρομή, η κρουαζιέρα, το δρομολόγιο, η πτήση, η αναχώρηση, η άφιξη, κλείνω εισιτήρια, κάνω κράτηση, το διαβατήριο, η βίζα.

*Правая:* το ταξιδιωτικό γραφείο / το τουριστικό πρακτορείο, το οργανωμένο ταξίδι / το πακέτο, το ξενοδοχείο (2–5 αστέρων) με πρωινό / ημιδιατροφή / πλήρη διατροφή, το δωμάτιο (μονόκλινο, δίκλινο, τρίκλινο, τετράκλινο), τα ενοικιαζόμενα δωμάτια, το κάμπινγκ.

*Примеры:* Στην πτήση Αθήνα–Πράγα… 200 επιβάτες… αναχώρηση… άφιξη… · Μπορείς… δρομολόγια των πλοίων… · …κλείσεις εισιτήρια… κράτηση… οργανωμένο ταξίδι (πακέτο). · …εκδρομή στο Ναύπλιο. · …κρουαζιέρα…
