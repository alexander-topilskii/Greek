## Словарь урока

### Чеклист (грамматика)

| Проверить | |
|----------|----------|
| πρέπει να / θέλω να / μπορώ να + να + ρήμα | фразы с να + правильная форма глагола |
| δε θα — να μην | в чём разница, на примере |
| ρεμπέτικο, λαϊκό, έντεχνο | три жанра: по одной фразе |
| τι περιλαμβάνει; — περιλαμβάνει (πακέτο) | пакетный тур: вопрос и ответ по списку |
