## Για δες

*(Στο βιβλίο: μεγάλος πίνακας — μέλλοντας με **θα** και υποτακτική με **να** · τύπος A. Στο βιβλίο υπάρχουν εικονίδια σημειώσεων σε ομάδες **–ξω** και **–ψω**.)*

### Κανονικά ρήματα — αλλαγές στο θεματικό

**–σω** · καταλήξεις **–νω**, **–ζω** → **σω**

| Ενεστώτας | θα + μέλλ. | να + υποτ. |
|---|------------|------------|
| πληρώνω | θα πληρώσω | να πληρώσω |
| αγοράζω | θα αγοράσω | να αγοράσω |

**–ξω** · καταλήξεις **–ζω**, **–γω**, **–χω**, **–χνω** → **ξω** *(σημείωση στο βιβλίο)*

| Ενεστώτας | θα + μέλλ. | να + υποτ. |
|---|------------|------------|
| αλλάζω | θα αλλάξω | να αλλάξω |
| ανοίγω | θα ανοίξω | να ανοίξω |
| προσέχω | θα προσέξω | να προσέξω |
| φτιάχνω | θα φτιάξω | να φτιάξω |

**–ψω** · καταλήξεις **–εύω**, **–πω**, **–φω**, **–βω** → **ψω** *(σημείωση στο βιβλίο)*

| Ενεστώτας | θα + μέλλ. | να + υποτ. |
|---|------------|------------|
| δουλεύω | θα δουλέψω | να δουλέψω |
| λείπω | θα λείψω | να λείψω |
| γράφω | θα γράψω | να γράψω |
| κλέβω | θα κλέψω | να κλέψω |

*Στο βιβλίο δεξιά — με το **Θέλω**: καταλήξεις **–ω**, **–εις**, **–ει**, **–ουμε**, **–ετε**, **–ουν(ε)**.*
