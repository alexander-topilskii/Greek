**16** *ενότητα*

**Πάμε βόλτα;** *(πάνω δεξιά)*

### 4. Διαλέγω το σωστό.

1. Θέλω (θέλω, πρέπει) να πάω βόλτα, αλλά __________ (μπορώ, πρέπει) να διαβάσω για το τεστ.
2. Δεν __________ (θέλω, μπορώ) να μαγειρέψω, γιατί χάλασε η κουζίνα.
3. __________ (έχω, λέω) να φάμε έξω απόψε. Συμφωνείς;
4. __________ (προσπαθώ, έχω) να πάω στην τράπεζα αύριο το πρωί.
5. Μου αρέσει το θέατρο, αλλά απόψε __________ (μπορώ, προτιμώ) να πάω σινεμά.
6. __________ (πάω, πάμε) να πιούμε ένα ποτό το βράδυ;

### 5. Διαλέγω και συμπληρώνω.

**Λέξεις:** να πάω, να τελειώσω, να στείλεις, να πάμε, να γυρίσω, να γράψουμε

1. — Θέλω **να πάω στο** θέατρο αύριο το βράδυ. Θα έρθεις; — Γιατί όχι; *(παράδειγμα στο βιβλίο)*
2. — Παιδιά, θα πάμε σινεμά απόψε. Μπορείτε; — Μπα. Έχουμε __________ τις ασκήσεις.
3. — Πάμε μια βόλτα μετά τη δουλειά; — Δεν μπορώ. Πρέπει __________ στο σπίτι νωρίς.
4. — Γιατί δεν πάμε για ψώνια τώρα; — Δε γίνεται. Έχω __________ τις δουλειές του σπιτιού.
5. — Σκεφτόμαστε __________ για μπάνιο το Σάββατο. Τι λέτε; — Ναι, ωραία ιδέα. Πού;
6. — Πάω στο ταχυδρομείο. Θέλεις τίποτα; — Ναι. Μπορείς __________ αυτό το γράμμα;

### 6. Συμπληρώνω όπως στο παράδειγμα.

1. — Θέλεις να πάμε βόλτα; — Ναι, θέλω **να πάμε**. *(παράδειγμα)*
2. — Θέλεις να πάω έξω; — Ναι, θέλω __________.
3. — Θέλετε να φύγετε τώρα; — Ναι, θέλουμε __________.
4. — Πρέπει να φύγεις αμέσως; — Ναι, πρέπει __________.
5. — Μπορεί να γυρίσεις αργά; — Ναι, μπορεί __________ αργά.
6. — Προτιμάς να γυρίσω νωρίς; — Ναι, προτιμώ __________ νωρίς.
