### Блок 1: Коммуникативные ситуации (Ролевые игры)

* **Ситуация 1: «Πάμε για δουλειά» — агентство, объявление, комикс**
* **Контекст:** Цели урока в рамке; объявление «ΜΙΚΡΕΣ ΑΓΓΕΛΙΕΣ»; реплики из комикса про отель.
* **Роль А (συνεντευξιαζόμενος / υποψήφιος):**
  1. Прочитайте вслух строку объявления: *ΥΠΑΛΛΗΛΟΣ για ρεσεψιόν με εμπειρία, ζητείται για ξενοδοχείο. Τηλ. 2810322398, 9–2 π.μ.*
  2. Озвучьте ответ героя в «облачке» комикса: *ΔΟΥΛΕΨΑ ΔΥΟ ΚΑΛΟΚΑΙΡΙΑ ΣΕ ΞΕΝΟΔΟΧΕΙΟ!*
  3. Добавьте одну фразу из рамки урока (*Τηλεφωνώ για μια αγγελία.* или *Έχετε εμπειρία;* — как вопрос к собеседнику).
* **Роль Β (εργοδότης / ξενοδοχείο):**
  1. Прочитайте вывеску: *HOTEL ΖΗΤΕΙΤΑΙ ΥΠΑΛΛΗΛΟΣ.*
  2. Задайте вопрос из комикса: *ΕΧΕΤΕ ΕΜΠΕΙΡΙΑ;*
  3. Коротко отреагируйте на ответ роли Α (например: *Πολύ ωραία. Στείλτε μας το βιογραφικό σας.*).
* 🎯 **Чек-лист для ситуации:**
  - *Τηλεφωνώ για μια αγγελία.* — Звоню по объявлению.
  - *Έχετε εμπειρία;* — Есть опыт?
  - *Στείλτε μας το βιογραφικό σας.* — Пришлите резюме.
  - *Ρώτα για τον μισθό.* — Спроси про зарплату.
  - *ΥΠΑΛΛΗΛΟΣ για ρεσεψιόν με εμπειρία, ζητείται για ξενοδοχείο. Τηλ. 2810322398, 9–2 π.μ.* — Текст объявления: ресепшн, опыт, телефон.
  - *Πινακίδα «HOTEL ΖΗΤΕΙΤΑΙ ΥΠΑΛΛΗΛΟΣ»* — Вывеска отеля: требуется сотрудник.
  - *ΕΧΕΤΕ ΕΜΠΕΙΡΙΑ;* — У вас есть опыт?
  - *ΔΟΥΛΕΨΑ ΔΥΟ ΚΑΛΟΚΑΙΡΙΑ ΣΕ ΞΕΝΟΔΟΧΕΙΟ!* — Ответ героя про два лета в отеле.

* **Ситуация 2: «Στείλτε μας το βιογραφικό σας» — телефонный разговор (B22)**
* **Контекст:** Фивос звонит по объявлению в «Нэа»; сотрудник уточняет имя, опыт, языки; адрес для CV.
* **Роль Α (Υπάλληλος):** Ведите по тексту; **Роль Β (Φοίβος):** отвечайте как соискатель.
* **Мини-линия «CV по пунктам» (после разговора):** Роль Β за 30–40 секунд называет вслух по гречески: *ονοματεπώνυμο*, *σπουδές*, *επαγγελματική εμπειρία*, *ξένες γλώσσες*, *Η/Υ* — по образцу анкеты Фивоса в главе.
* 🎯 **Чек-лист для ситуации:**
  - *Λέγετε, παρακαλώ.* — Слушаю вас.
  - *Καλημέρα. Τηλεφωνώ για την αγγελία σας στα «Νέα». Ενδιαφέρομαι για τη θέση υπαλλήλου ρεσεψιόν.* — Звоню по объявлению; интересует ресепшн.
  - *Βεβαίως. Το όνομά σας, κύριε;* — Ваше имя?
  - *Φοίβος Οικονόμου.* — Фивос Эконому.
  - *Κοιτάξτε, κύριε Οικονόμου. Είμαστε μια μεγάλη εταιρεία με οχτώ ξενοδοχεία σε όλη την Ελλάδα και ζητάμε ένα άτομο για μια πολύ υπεύθυνη θέση. Πείτε μου κάτι. Έχετε σχετική εμπειρία;* — Компания, восемь отелей; ответственная позиция; опыт?
  - *Να σας πω. Είμαι φοιτητής στη Σχολή Τουριστικών Επαγγελμάτων στο τελευταίο έτος.* — Студент туристической школы, последний курс.
  - *Μάλιστα. Ωραία. Απαντήστε μου, όμως, σε αυτό που σας ρωτάω. Έχετε προϋπηρεσία σε αυτό το επάγγελμα;* — Есть ли стаж по профессии?
  - *Βεβαίως. Δούλεψα δύο καλοκαίρια σε ένα ξενοδοχείο στη Χαλκιδική. Λατρεύω αυτή τη δουλειά και, επίσης, γνωρίζω αγγλικά, γαλλικά και γερμανικά.* — Два лета в отеле на Халкидиках; языки.
  - *Πολύ ωραία, κύριε Οικονόμου. Αυτό είναι σημαντικό. Προσέξτε με λοιπόν. Στείλτε μας το βιογραφικό σας σημείωμα στη διεύθυνση: Φιλοξενία, Ξενοδοχειακές Επιχειρήσεις Α.Ε., Χάνδακα 14, Ηράκλειο Κρήτης, Τ.Κ. 71414. Και γράψτε στον φάκελο: «Για την αγγελία». Εντάξει;* — Пришлите CV по адресу… пометка «по объявлению».
  - *Μάλιστα. Σας ευχαριστώ πολύ. Καλή σας μέρα.* — Спасибо, добрый день.
  - *Ενδιαφέρομαι για τη θέση υπαλλήλου ρεσεψιόν.* — Интересуюсь местом…
  - *Έχετε σχετική εμπειρία; / Έχετε προϋπηρεσία σε αυτό το επάγγελμα;* — Есть релевантный опыт / стаж?
  - *Κοιτάξτε. / Πείτε μου κάτι. / Προσέξτε με.* — Послушайте / скажите / обратите внимание.
  - *Στείλτε μας το βιογραφικό σας σημείωμα.* — Пришлите резюме.

* **Ситуация 3: «Η σειρά μου τώρα» — как искать работу (§2)**
* **Контекст:** Устные ответы на вопросы 1–5; списки «Ψάχνω για δουλειά».
* **Роль Α:** Задайте вопросы; **Роль Б:** ответьте и произнесите пункты списков.
* **Мини-диалог (рекрутёр — кандидат), лексика главы:** после ответов на вопросы 1–5 разыграйте 4–6 реплик.
  - *Α:* Έχετε προϋπηρεσία σε αυτή τη δουλειά;
  - *Β:* Ναι, δούλεψα δύο καλοκαίρια σε ξενοδοχείο. Γνωρίζω αγγλικά.
  - *Α:* Στείλτε μας το βιογραφικό σας. Και ρωτήστε για τον μισθό.
* 🎯 **Чек-лист для ситуации:**
  - *1. Δουλεύεις;*
  - *2. Πώς βρήκες τη δουλειά σου;*
  - *3. Έψαξες ποτέ για δουλειά στις αγγελίες; Ήταν εύκολο ή δύσκολο να βρεις δουλειά;*
  - *4. Είχες προϋπηρεσία;*
  - *5. Έστειλες βιογραφικό σημείωμα;*
  - *Διαβάζω τις μικρές αγγελίες*
  - *Ρωτάω φίλους και γνωστούς*
  - *Πάω στον ΟΑΕΔ (Οργανισμός Απασχόλησης Εργατικού Δυναμικού)*
  - *Ψάχνω στο Διαδίκτυο (Ίντερνετ)*
  - *Τηλεφωνώ*
  - *Στέλνω βιογραφικό*
  - *Πηγαίνω για συνέντευξη*

* **Ситуация 4: «Και ρώτα για τον μισθό…» — собеседование (B23)**
* **Контекст:** Фивос нервничает перед интервью; Никос и Панайотис советуют одежду и вопросы про график, зарплату, страховку.
* **Роль Α / Β / Γ:** Трое друзей — по репликам; затем второй блок «Πώς το λένε» со стикерами.
* 🎯 **Чек-лист для ситуации:**
  - *Έλα, μεγάλε. Κάθισε. Τι έχεις; Δε σε βλέπω και πολύ καλά…* — Садись; что с тобой?
  - *Άσε, ρε Νίκο. Πολύ άγχος.* — Ужасное волнение.
  - *Επειδή αύριο το πρωί πάω σε συνέντευξη για μια δουλειά σε ένα μεγάλο ξενοδοχείο.* — Завтра утром интервью в большом отеле.
  - *Πριν από λίγο. Δεν έχω ιδέα τι θα με ρωτήσουν. Θα τα πάω καλά; Είναι πολύ υπεύθυνη θέση.* — Не знаю, что спросят; ответственная должность.
  - *Καλά, ηρέμησε. Πήγαινε χωρίς φόβο, αυτό είναι το πιο σημαντικό.* — Успокойся; иди без страха.
  - *Ε, πες την αλήθεια. Μίλησέ τους για τις σπουδές σου, για τις γλώσσες που ξέρεις…* — Правда о учёбе и языках.
  - *Και μην πάς με το τζιν. Φόρεσε κάτι πιο σοβαρό.* — Не в джинсах; что-то серьёзнее.
  - *Βάλε ό,τι θέλεις, παιδί μου. Δείξε χαλαρός. Και ρώτησέ τους κι εσύ για τη δουλειά. Δεν είσαι κανένα παιδάκι. Θα έχεις σταθερό ωράριο; Θα έχεις ρεπό; Τι σύμβαση θα υπογράψεις;* — Будь спокойным; спроси и ты про работу.
  - *Βέβαια. Και ρώτα τους για τον μισθό, την ασφάλιση…* — Спроси про зарплату и страховку.
  - *Τι λέτε, ρε παιδιά; Σε λίγο θα τους ζητήσω και άδεια…* — Скоро попрошу и отпуск…
  - *Άσε, πολύ άγχος.* — Ох, сильное волнение.
  - *Δεν έχω ιδέα.* — Понятия не имею.
  - *Θα έχεις σταθερό ωράριο; — Θα έχεις ρεπό; — Τι σύμβαση θα υπογράψεις;* — Будет ли постоянный график? — Будут ли выходные? — Какой договор подпишешь?
  - *Καλά, ηρέμησε. — Δείξε χαλαρός.* — Ладно, успокойся. — Будь непринуждённым.
  - *Και ρώτα τους για τον μισθό, την ασφάλιση.* — И спроси про зарплату, страховку.
  - *Σε λίγο θα τους ζητήσω και άδεια.* — Скоро у них попрошу и отпуск.

* **Ситуация 5: Собеседование — вопросы §10 и роль 1.7**
* **Контекст:** «Η σειρά μου τώρα» §10; ролевая игра: звонок по сомнительному объявлению о зарплате.
* **Роль Α / Β:** Ответьте на вопросы §10; затем разыграйте сценарий по тексту 1.7.
* **Мини-диалог после списка:** кандидат уточняет условия; работодатель объясняет про пωλήσεις и ποσοστό.
  - *Α:* Ζητούν υπαλλήλους με 1.500 – 3.000 ευρώ τον μήνα. Τι ακριβώς κάνω;
  - *Β:* Ζητάμε ένα άτομο για πωλήσεις από το τηλέφωνο ή πόρτα-πόρτα. Δε θα έχει μισθό, αλλά θα παίρνει το 5% από τις πωλήσεις.
* 🎯 **Чек-лист для ситуации:**
  - *Πήγες ποτέ σε συνέντευξη για δουλειά;*
  - *Είχες άγχος;*
  - *Τι φόρεσες;*
  - *Τι σε ρώτησαν; Τι τους απάντησες;*
  - *Τι τους ρώτησες εσύ;*
  - *Τι έγινε μετά;*
  - *Διάβασα μια αγγελία στην εφημερίδα. Ζητούν υπαλλήλους που θα κερδίζουν 1.500 – 3.000 ευρώ τον μήνα. Δε γράφουν όμως τίποτα άλλο. Παίρνω τηλέφωνο και ρωτάω ό,τι θέλω.* — Роль Α: звонящий по объявлению.
  - *Μια κυρία / Ένας κύριος τηλεφωνεί για να ρωτήσει για την αγγελία που έχουμε βάλει στην εφημερίδα. Η εταιρεία μας ζητά ένα άτομο για πωλήσεις από το τηλέφωνο ή πόρτα-πόρτα. Δε θα έχει μισθό, αλλά θα παίρνει το 5% από τις πωλήσεις.* — Роль Β: сотрудник компании.

### Блок 2: Универсальная шпаргалка (Фразы-клише)

**Цели урока (вводная рамка)**

* *Τηλεφωνώ για μια αγγελία.* — Звоню по объявлению.
* *Στείλτε μας το βιογραφικό σας.* — Пришлите нам резюме.
* *Ρώτα για τον μισθό.* — Спроси про зарплату.

**Προστακτική — советы и указания (формы из упражнений главы)**

*Погода («Τι να κάνω;»):*
* *Άνοιξε το παράθυρο.* — Открой окно.
* *Ανάψτε το κλιματιστικό.* — Включите кондиционер.
* *Μην ανοίξετε τα παράθυρα.* — Не открывайте окна.
* *Πάρε ομπρέλα.* — Возьми зонт.
* *Φορέστε ζεστά ρούχα.* — Наденьте тёплую одежду.
* *Βάλε καπέλο.* — Надень шляпу/кепку.
* *Μην ξεχάσεις το μπουφάν.* — Не забудь куртку.

*Поиск работы (συμβουλές):*
* *Μάθε αγγλικά.* — Выучи английский.
* *Βρες μια άλλη δουλειά.* — Найди другую работу.
* *Ρωτήστε τον διευθυντή.* — Спросите директора.
* *Φόρεσε τα καλά σου ρούχα.* — Надень лучшую одежду.
* *Τηλεφώνησε και ρώτα.* — Позвони и спроси.

*Разрешение и запрет (пары из учебника):*
* *Αν θέλετε να καθίσετε, καθίστε. Αν δε θέλετε, μην καθίσετε.* — Если хотите сесть — сядьте; если нет — не садитесь.
* *πιες / μην πιεις* — выпей / не пей (ед. ч.).
* *ελάτε / μην έρθετε* — приходите / не приходите.
* *πήγαινε / μην πας* — иди / не иди.
* *μιλήστε / μην μιλήσετε* — говорите / не говорите.
* *φάε / μην φας* — ешь / не ешь.
* *περιμένετε / μην περιμένετε* — ждите / не ждите.

**Телефонный отклик на вакансию (B22 + первый «Πώς το λένε»)**

* *Ενδιαφέρομαι για τη θέση υπαλλήλου ρεσεψιόν.* — Интересуюсь местом сотрудника ресепшна.
* *Στείλτε μας το βιογραφικό σας σημείωμα.* — Пришлите резюме.
* *Μάλιστα. Ωραία. Απαντήστε μου, όμως, σε αυτό που σας ρωτάω. Έχετε προϋπηρεσία σε αυτό το επάγγελμα;* — Есть ли стаж по этой профессии?

**Перед интервью (B23 + стикеры)**

* *Άσε, πολύ άγχος.* — Ох, сильное волнение.
* *Και ρώτα τους για τον μισθό, την ασφάλιση.* — И спроси про зарплату, страховку.
* *Καλά, ηρέμησε. Πήγαινε χωρίς φόβο, αυτό είναι το πιο σημαντικό.* — Успокойся; иди без страха — это главное.

### Блок 3: Тематический словарь

**Работа и поиск**

| Слово на греческом | Перевод на русский |
|--------------------|--------------------|
| η αγγελία | объявление (о работе) |
| η μικρή αγγελία | мелкое объявление |
| η δουλειά | работа |
| το βιογραφικό σημείωμα | резюме, биографическая справка |
| η εμπειρία · η προϋπηρεσία | опыт · стаж |
| η συνέντευξη | собеседование |
| ο μισθός | зарплата |
| η ασφάλιση | страхование (соц.) |
| η θέση (εργασίας) | должность |
| ο εργοδότης · η εργοδότρια | работодатель |
| η σύμβαση εργασίας | трудовой договор |
| ο συνάδελφος | коллега |
| η μερική απασχόληση | частичная занятость |
| η απεργία | забастовка |
| η ανεργία | безработица |
| το επίδομα ανεργίας | пособие по безработице |
| ο ΟΑΕΔ | служба занятости (Греция) |

**Зарплата и удержания**

| Слово на греческом | Перевод на русский |
|--------------------|--------------------|
| τα μεικτά | «грязная» сумма (до удержаний) |
| τα καθαρά | «чистыми» (на руки) |
| οι κρατήσεις | удержания |
| τα ένσημα | отметки стажа (взносы в системе соцстраха) |
| οι υπερωρίες | сверхурочные |
| η αύξηση | повышение (зарплаты) |

**Офис и график**

| Слово на греческом | Перевод на русский |
|--------------------|--------------------|
| το ωράριο | график работы |
| το ρεπό | выходной (в графике) |
| η σύμβαση | контракт |
| η άδεια | отпуск |
| οι αργίες | праздничные выходные |
| η εταιρεία | фирма, компания |
| ο υπάλληλος | служащий, сотрудник |
| ο εργαζόμενος | работник |

**Глаголы и устойчивые выражения**

| Слово на греческом | Перевод на русский |
|--------------------|--------------------|
| ενδιαφέρομαι (για) | интересуюсь |
| ψάχνω (για δουλειά) | ищу (работу) |

| έχω άγχος | нервничаю, тревожусь |
