## Είμαι όλος αυτιά

*(Στο βιβλίο: εικονίδιο ακοής **B24**.)*

### 1.4

Η Ευγενία βρήκε τη δουλειά που ήθελε. Τώρα συζητά με τη διευθύντρια της εταιρείας. Ακούω τον διάλογο και βάζω ✓ στο σωστό ή στο λάθος.

| | Σωστό | Λάθος |
|--|--------|--------|
| 1. Η Ευγενία έχει εμπειρία στη δουλειά αυτή. | ✓ | |
| 2. Είναι η πρώτη φορά που η Ευγενία θα δουλέψει σε μια γνωστή εταιρεία. | | ✓ |
| 3. Η Ευγενία πρέπει να μάθει ηλεκτρονικούς υπολογιστές. | | ✓ |
| 4. Η Ευγενία θα ξεκινήσει την καινούρια της δουλειά τη Δευτέρα. | — | — |
| 5. Ο μισθός της θα είναι 1.100 ευρώ μεικτά. | | ✓ |
