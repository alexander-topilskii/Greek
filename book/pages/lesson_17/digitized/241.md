## Για δές

*(Στο βιβλίο: πίνακες προστακτικής με αντωνυμίες — χρώματα πράσινο, ανοικτό, πορτοκαλί, μπλε· πάνω από τις στήλες, όπου υπάρχουν, αριθμοί συλλαβών 3–2–1.)*

### Προστακτική + αντικείμενο / αντωνυμία

| *(ενικό πλήρες)* | *(ενικό αντωνυμία)* | *(πληθυντικό πλήρες)* | *(πληθυντικό αντωνυμία)* |
|-------------|-------------------|-------------------|---------------------|
| Κοίταξε εμένα. | Κοίταξέ με. | Κοίταξε εμάς. | Κοίταξέ μας. |
| Άνοιξε τον υπολογιστή. | Άνοιξέ τον. | Άνοιξε τους υπολογιστές. | Άνοιξέ τους. |
| Διάβασε την αγγελία. | Διάβασέ την. | Διάβασε τις αγγελίες. | Διάβασέ τες / τις. |
| Άλλαξε το βιογραφικό σου. | Άλλαξέ το. | Άλλαξε τα βιογραφικά. | Άλλαξέ τα. |

### Συμπτυγμένη προστακτική

| *(πλήρης)* | *(με αντωνυμία)* |
|--------|------------------|
| Κλείσε τον υπολογιστή. | Κλείσ’ τον. |
| Γράψε τις αγγελίες. | Γράψ’ τες / τις. |
| Στείλε το βιογραφικό. | Στείλ’ το. |

### Έμμεσα αντικείμενα

| *(με **σε** + αντικείμενο)* | *(με αντωνυμία)* |
|------------------------|------------------|
| Τηλεφώνησε σε εμένα. | Τηλεφώνησέ μου. |
| Τηλεφώνησε στον Ερβίν. | Τηλεφώνησέ του. |
| Τηλεφώνησε στην Αρλέτα. | Τηλεφώνησέ της. |
| Τηλεφώνησε σε εμάς. | Τηλεφώνησέ μας. |
| Τηλεφώνησε στους φίλους σου. | Τηλεφώνησέ τους. |
| Τηλεφώνησε στις κοπέλες. | Τηλεφώνησέ τους. |
| Τηλεφώνησε στα παιδιά. | Τηλεφώνησέ τους. |
