### Чеклист (грамматика)

| Проверить | |
|----------|----------|
| ήμουν / ήσουν / ήταν — είμαι | прошлое / настоящее «быть» |
| πήγα / πήγες — πάω | аорист «идти, ездить» |
| χτες + αόριστο | вчера + аорист (αγόρασα, έφτασα, ταξίδεψα) |
| μίλησα, ξύπνησα, πέρασα | аорист типа B (β) |
| έκανα, έφαγα, είδα · τους είδα | нерегулярные: κάνω, τρώω, βλέπω |
