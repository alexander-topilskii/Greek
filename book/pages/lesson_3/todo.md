## Контент и учебная ценность (в приоритете)

### `essence_3/essence_3.md`

**Что уже хорошо для изучения и отработки**

- Охвачены цели урока из блока «Στόχοι»: семья, **ποιος/ποια/ποιο**, **τον/την/το λένε**, профессия (**τι δουλειά κάνεις;**), таблица **δουλεύω / σπουδάζω**, семейный статус и род профессий (пары муж./жен. и общий род).
- Диалоги **A21** и **A22** даны полностью; блок «Για δες» по **ποιος/ποια** и **πώς … λένε** (кроме пары «как вас зовут» — см. ниже); эталоны для текстов **Α, Β, Γ**, верно/неверно по **A21**, упр. 6, 8, 9, «Τι λείπει;», семейного дерева; модель **Τη λένε … Είναι … και δουλεύει …**
- Итоговая лексическая таблица объёмная и в целом соответствует идее «всё из главы в одном месте».

**Пробелы и риски**

- Конструкция **Πώς σε λένε;** — **Με λένε …** из раздела «Για δες» (стр. 40 в оцифровке) в конспекте не выделена: ни в таблице готовых фраз, ни в итоговом словаре (в файле нет вхождений **Πώς σε λένε** и **Με λένε**). Это базовая реплика для знакомства; без неё слабее контраст «как зовут другого» (**πώς τον λένε**) и «как меня зовут».
- Слово **τηλέφωνο** есть только в примере к **Πώς το λένε στα ελληνικά;**; в итоговой таблице раздела со словарём его нет — при строгой трактовке правила «все уникальные слова материала» стоит добавить строку **το τηλέφωνο** | телефон (или убрать пример из грамматики, если слово не считается активным).
- **Аудирование A23** («Είμαι όλος αυτιά»): в оцифровке есть задание с вариантами 2–5; в конспекте нет компактного ключа для самопроверки (в лексике есть заголовок и имена, но не ответы «какой вариант верен»). Для повторения без аудиозаписи полезно дать короткий ключ или явно указать, что блок рассчитан на прослушивание.
- **Упр. 4** (стр. 41): пункты 5–10 завязаны на портреты; в конспекте есть образцы для 1–4 и список фамилий в оцифровке, но нет эталона «под номером N — это …» для работы без скана. Имеет смысл короткий ключ по порядку издания или пометка, что нужен визуальный ряд.
- **Упр. 16** (стр. 47): в оцифровке дан только пример **Η Λίλιαν είναι γιατρός.**; остальные строки пустые. Для автономной отработки шаблона «… είναι …» полезно добавить эталонные предложения по строкам таблицы (в рамках лексики урока), по аналогии с упр. 9.
- **Фонетика 1.2–1.4** (τ/θ, δ/ντ, σ/ζ в топонимах): в оцифровке только формулировка; в конспекте нет замкнутых примеров с ответами. По [essence-generation.md](../../../docs/essence-generation.md) для фонетики желательны заполненные слова или ключ, чтобы не сводить повторение к чтению длинных списков топонимов без опоры.

**Мелкое оформление**

- В блоке «Род профессий» сломана разметка: у **ηθοποιός** оборваны звёздочки `**` (строки с **υπάλληλος** / **ηθοποιός**); перенос **άνεργος /** на новую строку портит список — лучше одна строка или маркированный список.

### `lesson_voice_3/voice_lesson_3.md`

**Что уже хорошо**

- Тематически охвачены главные линии главы: **A21**, «Για δες», семейный статус, **A22**, мини-презентация по упр. 9, карточки **1.5**, фрагменты лексики и таблица профессий — близко к [lesson_3_digitized.md](lesson_digitized/lesson_3_digitized.md).

**Существенные отклонения от стандарта и от промпта**

- Файл почти целиком состоит из больших фрагментов оцифровки, обёрнутых в fenced code blocks (тройные обратные кавычки с меткой `text`), с непереработанным текстом: заголовки учебника, пометки *(εικονίδιο …)*, *(πορτοκαλί κουτιά)*, таблицы как в пособии. Это противоречит [voice_lesson_generation.md](../../../docs/promts/voice_lesson_generation.md): запрет технической разметки исходника, запрет копировать куски целиком, требование синтеза; чеклист [voice-lesson-from-digitized.md](../../../docs/voice-lesson-from-digitized.md) — без цветовых пометок, без оборачивания всего материала в код.
- Структура ситуаций не совпадает с шаблоном промпта: у каждой ситуации должны быть явные **Контекст**, **Роль А**, **Роль Б** (пошагово по-русски) и **Чек-лист** с 3–5 фразами **греч. — рус.** без отсылки «см. таблицу в цитате».
- **Блок 2** должен быть единой шпаргалкой по смысловым группам, формат **греческая фраза** — *русский перевод*; сейчас это снова «срез оцифровки» в коде.
- **Блок 3** по промпту — 4–6 таблиц «греч. | рус.» с артиклями у существительных; сейчас лексика в виде цитат без системной русификации и без артиклей во всех строках списков.
- Нет хлебных крошек и узкой таблицы «Быстрые ссылки» с [content_3.md](content_3.md) после заголовка `#` ([voice-lesson-from-digitized.md](../../../docs/voice-lesson-from-digitized.md), [AGENTS.md](../../../AGENTS.md)).
