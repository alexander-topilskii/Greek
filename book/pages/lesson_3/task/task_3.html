Урок 3.

Ситуация 1: Полное знакомство

Описание: Два человека впервые говорят о себе: имя, работа, семья и дети.

Ход диалога:

  1. Πώς σε λένε; → Με λένε …
  2. Τι δουλειά κάνεις; (Τι δουλειά κάνετε;) → Είμαι … (Δουλεύω …, при необходимости Δουλεύω στην …).
  3. Είσαι παντρεμένος; / παντρεμένη; → Ναι / Όχι, Είμαι ελεύθερος / ελεύθερη.
  4. Έχεις παιδιά; → ответ (Ναι, έχω … / Όχι).
  5. B задаёт тем же темам вопросы к A.

Подсказки:

  • επάγγελμα из урока: γιατρός, ηλεκτρολόγος, καθηγητής, καθηγήτρια …
  • Τι δουλειά κάνεις; · Πού δουλεύεις;

Ситуация 2: Фотография («кто это?» и имена в винительном падеже)

Описание: Смотрите на фото и отрабатываете ποιος / ποια / ποιο и формы Τον λένε … / Τη λένε ….

Ход диалога:

  1. A: Ποιος είναι αυτός; / Ποια είναι αυτή; / (Ποιο είναι αυτό; если предмет)
  2. B: Ο … · Η … · Το …
  3. A: Πώς τον λένε; / Πώς τη λένε;
  4. B: Τον λένε Γιάννη. · Τη λένε Ελένη. (имя как в диалогах учебника)
  5. По желанию: расскажите о работе третьего лица по образцу Είναι γιατρός. / Δουλεύει στην Ελλάδα.

Подсказки:

  • согласование рода вопроса и ответа.
  • аккуратно с именами: Τον λένε Γιώργο. (форма после λένε).

Ситуация 3: Семья

Описание: Речь о οικογένειά μου, родителях, брате / сестре, детях.

Ход диалога:

  1. Ποιος είναι αυτός; → Ο πατέρας μου. (Ο μπαμπάς μου.)
  2. Ποια είναι αυτή; → Η μητέρα μου.
  3. Краткий рассказ по цепочке: муж / жена (ο άντρας, η γυναίκα), ο γιός, η κόρη.
  4. B переспрашивает о семье A.

Подсказки:

  • Η οικογένειά μου · родственники из конспекта.
  • Ο αδελφός μου / Η αδελφή μου

Ситуация 4: Личная жизнь («с кем живёшь»)

Описание: Семейное положение + Μένω με … (перекликается с уроком 2).

Ход диалога:

  1. Είσαι παντρεμένος; / παντρεμένη;
  2. Если нет брака — Είμαι ελεύθερος / ελεύθερη (χωρισμένος — по желанию).
  3. Μένεις με …; / Μένω με τον / την / τη φίλη μου.
  4. Обмен теми же темами с партнёром.

Подсказки:

  • Μένω με τη φίλη μου. (как фраза учебника)
  • Не смешивайте Είσαι παντρεμένος (м. пр.) и παντρεμένη (ж. пр.) в одном коротком ответе без согласования.

Ситуация 5: Только профессия и место

Описание: Короткий обмен только о работе и (опционально) стране или городе.

Ход диалога:

  1. Τι δουλειά κάνεις; → Είμαι …
  2. Πού δουλεύεις; → Δουλεύω στην Αθήνα. / στην Ελλάδα. (повтор темы топонимов + предлога σε со στην / στον.)
  3. B отвечает и задаёт те же два вопроса.

Подсказки:

  • Τι δουλειά κάνετε; при вежливом тоне.
  • Профессии с таблицы урока; при сомнении в роде — опора на образцы Είμαι γιατρός.
