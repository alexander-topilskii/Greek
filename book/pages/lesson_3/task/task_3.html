Урок 3.

🧩 Ситуация 1: Полное знакомство + семья + личная жизнь

📍 Описание

Два человека знакомятся и подробно обсуждают: имя, профессию, место работы, семейное положение, партнёра (муж/жена), с кем живут и есть ли дети.

🔄 Ход диалога

  1. A спрашивает, как зовут B
  2. B отвечает
  3. A спрашивает, кем работает B
  4. B отвечает и добавляет, где работает
  5. A спрашивает, женат/замужем ли B
  6. B отвечает
  7. A спрашивает, кто муж/жена B
  8. B отвечает (имя + кем работает)
  9. A спрашивает, с кем живёт B
  10. B отвечает
  11. A спрашивает, есть ли у B дети
  12. B отвечает
  13. B задаёт все те же вопросы A

💡 Подсказки

  • Πώς σε λένε; → Με λένε …
  • Τι δουλειά κάνεις; / Τι δουλειά κάνετε;
  • Είμαι … / Δουλεύω ως … / Δουλεύω στην …
  • Πού δουλεύεις;
  • Είσαι παντρεμένος; / παντρεμένη;
  • Ναι, είμαι / Όχι, δεν είμαι
  • Είμαι ελεύθερος / ελεύθερη
  • Ποιος είναι ο άντρας σου; / Ποια είναι η γυναίκα σου;
  • Πώς τον λένε; / Πώς τη λένε;
  • Τον λένε … / Τη λένε …
  • Τι δουλειά κάνει;
  • Είναι … / Δουλεύει ως …
  • Μένω με …
  • Μένω με τον άντρα μου / τη γυναίκα μου
  • Έχεις παιδιά; → Ναι, έχω … / Όχι

🧩 Ситуация 2: Фото + третье лицо + семья + «кто это»

📍 Описание

Вы смотрите на фотографию и подробно обсуждаете: кто это, как зовут, кем работает, член семьи или нет, а также простые предметы/животные (например: «это мой кот»).

🔄 Ход диалога

  1. A спрашивает, кто/что изображено (мужчина, женщина, ребёнок, предмет, животное)
  2. B отвечает
  3. A спрашивает, как его/её/это называют
  4. B отвечает
  5. A спрашивает, кем он/она работает (если это человек)
  6. B отвечает
  7. A уточняет, кто это для B (муж, жена, ребёнок, друг и т.д.)
  8. B отвечает
  9. Добавить пример с предметом/животным (например: «это мой кот, его зовут…»)
  10. B задаёт аналогичные вопросы A

💡 Подсказки

  • Ποιος είναι αυτός; / Ποια είναι αυτή; / Ποιο είναι αυτό;
  • Ο … / Η … / Το …
  • Είναι ο άντρας μου / η γυναίκα μου / το παιδί μου

  • Είναι φίλος μου / φίλη μου

  • Πώς τον λένε; / Πώς τη λένε; / Πώς το λένε;

  • Τον λένε … / Τη λένε … / Το λένε …

  • Τι δουλειά κάνει;

  • Είναι … / Δουλεύει ως … / Δουλεύει στην …

  • Это мой кот:

    • Είναι το παιδί μου (для ребёнка)
    • Είναι ο άντρας μου / η γυναίκα μου
    • Είναι το … μου

🧩 Ситуация 3: Работа (углублённая + вариации)

📍 Описание

Подробный разговор о работе с использованием разных формулировок: «я — …», «я работаю …», «моя работа — …», а также обсуждение работы другого человека.

🔄 Ход диалога

  1. A спрашивает, кем работает B
  2. B отвечает несколькими способами
  3. A спрашивает, где работает B
  4. B отвечает
  5. A спрашивает о работе мужа/жены B
  6. B отвечает
  7. B задаёт те же вопросы A

💡 Подсказки

  • Τι δουλειά κάνεις; / Τι δουλειά κάνετε;

  • Πού δουλεύεις;

  • Είμαι …

  • Δουλεύω ως …
  • Η δουλειά μου είναι …

  • Δουλεύω στην …

  • Τι δουλειά κάνει ο άντρας σου;

  • Τι δουλειά κάνει η γυναίκα σου;
  • Είναι … / Δουλεύει ως …

🧠 Примеры (близко к учебнику)

  • Είμαι γιατρός.
  • Είμαι καθηγητής / καθηγήτρια.
  • Είμαι αρχιτέκτονας.

  • Είμαι προγραμματιστής.

  • Δουλεύω ως αρχιτέκτονας. (я работаю архитектором)

  • Δουλεύω ως προγραμματιστής.

  • Δουλεύω ως καθηγήτρια. (женщина)

  • Η δουλειά μου είναι αρχιτέκτονας.

  • Η δουλειά μου είναι προγραμματιστής.

  • Δουλεύω στην Ελλάδα.

  • Δουλεύω στην Αθήνα.

  • Ο άντρας μου είναι γιατρός.

  • Η γυναίκα μου είναι καθηγήτρια.
  • Ο άντρας μου δουλεύει ως μηχανικός.
