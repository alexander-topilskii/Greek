## 📌 Тема и цели

- **Γεια σου / Γεια σας** — неформально / вежливо или мн. ч.
- **Είμαι ο/η …** — я (муж./жен.); вопросы **Είσαι …;** **Είστε …;**
- **Από πού είσαι; / Από πού είστε;** — откуда ты / вы?
- **Είμαι από τον/την/το …** — я из … (с артиклем в вин. падеже)
- **Πώς σε λένε; / Πώς λέγεστε;** — как тебя / вас зовут?
- **Με λένε …** — меня зовут …
- Числа **μηδέν – δέκα**; греческие **; . , !**

## 1) Ключевые слова

### Глагол **είμαι** (настоящее)

| лицо       | форма   |
|------------|---------|
| εγώ        | είμαι   |
| εσύ        | είσαι   |
| αυτός/ή/ό  | είναι   |
| εμείς      | είμαστε |
| εσείς      | είστε   |
| αυτοί/ές/ά | είναι   |

**Практика:** три фразы: *Εγώ είμαι ο/η …*, *Εσύ είσαι από …;*, *Εμείς είμαστε από …*.

### Приветствия и обращение

| ελληνικά   | по-русски                              |
|------------|----------------------------------------|
| γεια σου   | привет / пока (ты)                     |
| γεια σας   | здравствуйте / до свидания (вы, вежл.) |
| καλημέρα   | доброе утро / день                     |
| καλησπέρα  | добрый вечер                           |
| καληνύχτα  | спокойной ночи                         |
| χαίρω πολύ | рад(а) познакомиться                   |
| κύριε …    | господин … (обращение)                 |
| κυρία …    | госпожа …                              |
| παρακαλώ   | пожалуйста                             |
| ευχαριστώ  | спасибо                                |
| συγνώμη    | извините                               |

### Страны (с артиклем для **από …**)

| род | пример                                                                                       | по-русски        |
|-----|----------------------------------------------------------------------------------------------|------------------|
| ο   | ο Καναδάς — **από τον Καναδά**                                                               | Канада           |
| η   | η Ισπανία — **από την Ισπανία**                                                              | Испания          |
| η   | η Ιταλία, η Κίνα, η Ρωσία, η Ουκρανία, η Αλβανία, η Βουλγαρία, η Αγγλία, η Ελλάδα, η Τουρκία |                  |
| το  | το Μεξικό, το Βέλγιο — **από το …**                                                          | Мексика, Бельгия |
| ο   | ο Λίβανος — **από τον Λίβανο**                                                               | Ливан            |

### Имя: вопросы и ответы

| ελληνικά                   | по-русски                    |
|----------------------------|------------------------------|
| Πώς σε λένε;               | Как тебя зовут?              |
| Με λένε …                  | Меня зовут …                 |
| Πώς λέγεστε;               | Как вас зовут? (вежл.)       |
| Πώς τον λένε; / Τον λένε … | Как его зовут? / Его зовут … |
| Πώς τη λένε; / Τη λένε …   | Как её зовут? / Её зовут …   |
| Από 'δώ ο/η …              | Познакомьтесь: …             |

### Существительные (урок)

| ελληνικά     | по-русски               |
|--------------|-------------------------|
| η τάξη       | класс                   |
| το μάθημα    | урок                    |
| η γραμματεία | канцелярия, секретариат |
| η φίλη       | подруга                 |
| τα παιδιά    | дети                    |
| το παιδί     | ребёнок                 |

### Числа 0–10

| цифра | ελληνικά | примечание   |
|-------|----------|--------------|
| 0     | μηδέν    |              |
| 1     | ένα      |              |
| 2     | δύο      | также **δυο** |
| 3     | τρία     |              |
| 4     | τέσσερα  |              |
| 5     | πέντε    |              |
| 6     | έξι      |              |
| 7     | εφτά     | также **επτά** |
| 8     | οχτώ     | также **οκτώ** |
| 9     | εννιά    | также **εννέα** |
| 10    | δέκα     |              |

**Практика:** проговорите ряд **μηδέν, ένα, … δέκα**; напишите цифрами словами: 3, 7, 10.

### Знаки препинания (греческие)

| Знак  | Название / пример                               |
|-------|-------------------------------------------------|
| **;** | вопросительный знак (точка с запятой вверху) — **Εεε;** |
| **!** | восклицание (**Αααα!**)                         |
| **,** | запятая (**Ελένη, έλα.**)                       |
| **.** | точка (**Εντάξει.**)                            |

### Ударение (**τόνος**)

В слове ударение — **на одном из трёх последних слогов**. Различайте смысл по ударению:

| Слово 1   | Слово 2   | Заметка                          |
|-----------|-----------|----------------------------------|
| **πότε**  | **ποτέ**  | когда? / никогда                 |
| **τάξη**  | **ταξί**  | класс / такси                    |
| **σχολείο** | **σχόλιο** | школа / замечание              |
| **γράφει** | **γραφή** | пишет / письмо, написание       |

**Практика:** расставьте ударение: **Μαρία, Γιώργος, Γιάννης, όχι, καλημέρα, γεια σου**.

## 2) Произношение: гласные и сочетания

### Звуки [a] [e] [i] [o] [u] (блок «Για δες»)

*Кратко: одна буква или сочетание соответствует устойчивому звуку; **αι** и **ε** дают [e], **ι, η, υ, ει, οι, υι** — [i] и т. д.*

| Звук    | Буквы / сочетания                                        | Примеры для чтения       |
|---------|----------------------------------------------------------|--------------------------|
| [a]     | α                                                        | Αλεξάνδρα, Αθήνα         |
| [e]     | ε, αι                                                    | Ελένη, Αιμιλία           |
| [i]     | ι, η, υ, ει, οι, υι                                      | Ηλίας, Ειρήνη, οικονομία |
| [o]     | ο, ω                                                     | Όλγα, Ρωξάνη             |
| [u]     | ου                                                       | Ουρανία                  |
| [f]/[v] | φ; **αυ, ευ** перед глухим/звонким согласным или гласной | см. правило ниже         |

**Правило αυ, ευ:** перед **κ, π, τ, φ, θ, χ, σ, ξ, ψ** — слышится **[af]/[ef]**; перед гласной и звонкими согласными —
**[av]/[ev]**. Потренируйтесь: **Αυστραλία, αύριο, ευχαριστώ, εύκολο, Ευρώπη, ευτυχία, αυτοκίνητο, αυτός, Λευκορωσία, Μαυριτανία, ευρώ** — разложите по типам [af]/[av]/[ef]/[ev] вслух.

### Сочетания μπ, ντ, γγ/γκ, τσ, τζ

| Написание | Звук         | Примеры          |
|-----------|--------------|------------------|
| μπ        | [b] или [mb] | μπάσκετ, ομπρέλα |
| ντ        | [d] или [nd] | ντίσκο, Αντρέας  |
| γγ, γκ    | [g] или [ng] | γκαράζ, φεγγάρι  |
| τσ        | [ts]         | πίτσα            |
| τζ        | [dz]         | τζιν             |

**Чтение по словам (типовой список «Διαβάζω»):** **μπανάνα, ντομάτα, τσάι, τζατζίκι, πίτσα, ντίσκο, ομπρέλα, γκαράζ, φεγγάρι** — проговорите вслух.

## 3) Словосочетания и реплики

### Диалоги: неформально и на «вы»

**В кафе (несколько подруг):**  
— **Γεια σας.**  
— **Γεια σου, Μελέκ. Από 'δώ η Βέρα.**  
— **Γεια σου, Βέρα.**  
— **Γεια σου, Μελέκ.**

**Уважительно к старшему:**  
— **Γεια σας, κύριε Αντρέα.**  
— **Γεια σου, Παναγιώτη μου.**

*Обращение по имени:* к мужчине вежливо часто **κύριε + имя в звательном** (**κύριε Αντρέα**); ласково к молодому — **Παναγιώτη μου** (звательный **-η** у **-ης**).

### Улица / кафе (сцена знакомства)

- **Γεια σας.** — Здравствуйте.
- **Γεια σου, … Από 'δώ η …** — Привет, … это …
- **Είμαι ο/η …** — Я …
- **Εσύ είσαι η …;** — Ты …?
- **Όχι, δεν είμαι … Είμαι …** — Нет, я не … Я …
- **Έλα! Εδώ είμαστε.** — Иди! Мы здесь.

### Имя и страна (в классе / у двери)

- **Καλημέρα. Γεια, είμαι ο … Πώς σε λένε;**
- **Με λένε …**
- **Είσαι από την …;** — Ты из …?
- **Όχι. Είμαι από την … Εσύ από πού είσαι;**
- **Από την …**
- **Καλημέρα σας. Είμαι για το μάθημα.**
- **Πώς λέγεστε, παρακαλώ;**
- **Από πού είστε;**
- **Εδώ είναι η τάξη σας.**

### Чтение: вывески и слова недели

Слова для чтения: **καφέ, ταβέρνα, αδέλφια, επτά**, **Παρασκευή, Σάββατο, αργίες**, **ανοίγουμε, μετά τις …, το πρωί**.

**Образцы фраз:** **Ανοίγουμε μετά τις 9.** / **Αργίες: Παρασκευή, Σάββατο.**

**Практика:** составьте одну фразу с числом и одну с днём недели по образцу вывески.
