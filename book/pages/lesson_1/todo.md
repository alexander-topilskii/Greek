## Контент и самодостаточность (`essence_1.md`)

- **Оформление по [docs/essence-generation.md](../../../docs/essence-generation.md):** после заголовка нет **хлебных крошек** до `Readme` → `book/pages` → `content_1.md` → текущий файл и узкой таблицы **«Быстрые ссылки»** (раздел 2 документа). Имеет смысл выровнять с остальными уроками.
- **Нарушение автономности (критично):** в блоке про согласные есть ссылка на сырьё `[digitized/17.md](../digitized/17.md)` — такие ссылки в конспекте **запрещены**; расширенный список слов для чтения нужно **встроить в конспект** или убрать отсылку.
- **Итоговая таблица лексики:** сверить строки с текстом единицы в `lesson_1_digitized.md`. По выборочной проверке в оцифровке **не встречаются**, тогда как они есть в таблице — риск «шума» и путаницы при голосовой отработке (промпт в [docs/ai_voice_promt.md](../../../docs/ai_voice_promt.md) опирается на итоговую таблицу). Кандидаты на пересмотр: `Ξέρω`, `θέλω`, `μάλιστα`, `ισπανικά`, `κινηματογράφος`, `Πέδρο`, `Μάρκος`, `Φωτεινή`, `ωραία`, `τράπεζα`, `Πέμπτη`, `Σάββατο` (рядом с корректным `Πάμπλο`) — уточнить источник, удалить лишнее или дать опорный контекст из материала урока.
- **Практика без пропусков:** в оцифровке есть диалог **2.2 Συμπληρώνω** (Мелек — Пабло — Никос) с пустыми репликами; в конспекте **нет заполненного эталона** для самопроверки. Для цели «готовый материал для отработки» стоит добавить **полный текст** этого обмена (как минимум в разделе эталонов).
- **Фонетика / орфография:** правила **αι, αυ, ευ** и таблица **μπ, ντ, γκ** даны, но упражнения с пропусками (**β ή μπ;**, **γ ή γκ**, **δ ή ντ** и т.д.) **без ключей** — читатель не может проверить написание. По [docs/essence-generation.md](../../../docs/essence-generation.md) для фонетики желательны **замкнутые примеры**; имеет смысл дать **сжатый ключ** (слово целиком) или скорректировать формулировки целей в начале файла.

## Голосовой урок (`voice_lesson_1.md`)

- **Автономность ([docs/voice-lesson-from-digitized.md](../../../docs/voice-lesson-from-digitized.md)):** в ситуации 4 для роли Б указано «по цепочке диалога **A10 в оцифровке**» — прямая отсылка к сырью и номеру в источнике; заменить на **нейтральную инструкцию** (ответить по репликам диалога у класса без слов «оцифровка», «A10»).
- **Метаформулировки:** «**как в учебном диалоге**» (ситуация 2) для читателя звучит как отсылка к пособию; лучше описать **ситуацию и речевой этикет** без этого.
- **Баланс с главой:** по `lesson_1_digitized.md` большой блок — **фонетика, ударение, числа, вывески**; в `voice_lesson_1.md` почти весь объём — коммуникативные сценарии. Для отработки **урока 1 целиком** стоит **расширить блок 2–3**: опорные фразы для чтения (**αυ/ευ**, **μπ/ντ/γκ**, числа **0–10**, пары с **τόνος**) в духе [docs/promts/voice_lesson_generation.md](../../../docs/promts/voice_lesson_generation.md), не выходя за лексику главы.
