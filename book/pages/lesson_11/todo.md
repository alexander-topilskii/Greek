## Контент и учебная ценность (в приоритете)

### `essence_11/essence_11.md`

**Что уже хорошо для изучения и отработки**

- Три больших диалога (**B1, B2, B3**) даны полностью — это ядро коммуникативной темы (объявление, звонок, просмотр).
- В **грамматике** вынесена таблица **именительный / родительный** и блок про **предлоги места**; есть **сравнение** (опора на прилагательное «дорогой») и задел по **Γ2** (в эталонах упр. 8).
- **Эталоны** закрывают ряд типовых заданий (заполнение про Ставра, верно/неверно по B2 и B3, формы Γ2).
- **Итоговая лексика** очень полная (включая заголовки и формулировки заданий) — удобно для обзора и самопроверки «всё ли прошёл».

**Пробелы и риски для читателя без главы**

- Текст в **жёлтой рамке** (проживание у друзей, университет Фессалии, высокие **ενοίκια**, разговорный образ про «огонь») в оцифровке главы есть, в конспекте **не перенесён** — отдельный связный текст для чтения и лексики; имеет смысл добавить в раздел **готовых фраз / эталонов** с кратким пояснением.
- Раздел главы про **календарь и даты** (таблица месяцев, микродиалоги про срок освобождения квартиры, рандеву, **ενοίκιο**, «какое число»): в конспекте по сути сведён к отсылке на файл страницы — для автономного повторения лучше **встроить** таблицу дат и несколько готовых реплик из главы.
- Правило **ударения в родительном** (на моделях **κύριος**, **δωμάτιο**, **σπίτι**) в главе развернуто; в конспекте читатель в основном видит ссылку на `digitized` — для отработки темы полезно **кратко воспроизвести** схему и примеры в разделе **грамматики**.
- Часть упражнений опирается на **картинки** (3–4, 9, разметка 2.3 и др.): эталоны местами даны частично или только «опорные формы» — это нормально как лимит, но для самостоятельной работы можно явно пометить «нужна картинка из материалов урока» **одной нейтральной фразой** без ссылки на файл сырья или добавить ещё 1–2 микро-эталона там, где смысл однозначен из текста.

### `lesson_voice_11/voice_lesson_11.md`

**Что уже хорошо**

- Охвачены главные **сюжетные линии** главы: кафенейон + объявление, опорные фразы, звонок (**B2**), просмотр (**B3**), роль у **риелтора** (2.4).
- **Шпаргалка** и словарь дают быстрый вход в лексику жилья и ориентиров.

**Пробелы для разговорной отработки**

- Нет компактного блока по **датам и срокам** (календарь и микродиалоги из соответствующего раздела главы) — пригодилось бы несколько реплик в шпаргалке или отдельная ситуация.
- **Предложения места** из сцены у **γραφείο** (пан/над/под/внутри/снаружи) и **маршрут** из **3D-карты** (стр. 140) почти не попали в чек-листы — для урока про «где живу / как пройти» это заметный пробел по сравнению с главой.
- **Словарь (блок 3)** узкий относительно объёма урока: нет отдельных строк для **μεσίτης**, **ιδιοκτήτης**, **ρετιρέ**, **τεσσάρι**, **γκαρσονιέρα** и т.д., хотя часть встречается в сценариях — для **отработки** лучше расширить таблицы или дать вторую группу «квартира по типам и сделка».
- Ситуация 5: сценарии даны **цитатой из методички** («я иду к риелтору…»), а не разбиты на **живые шаги** с готовыми греческими репликами — слабее, чем ситуации 1–4, плюс **битая строка** чек-листа (см. ниже).

## Формат и правила репозитория

### `essence_11/essence_11.md`

1. **Ссылки на сырьё (`digitized/`)** — в теле конспекта есть markdown-ссылки на `../digitized/*.md`. По [essence-generation.md](../../../docs/essence-generation.md) (п. 1.2) их нужно убрать, перенеся смысл в файл.

2. **Шапка** — нет хлебных крошек и узкой таблицы **«Быстрые ссылки»** с `content_11.md` (раздел 2 в essence-generation, [agents.md](../../../AGENTS.md)).

3. **«Сверка по скану»** в эталонах — лучше нейтральный ориентир по странице или только текст эталона.

4. **Порядок разделов** — после итоговой лексической таблицы идёт **«Продуктивное задание»** со ссылками на `digitized/`; итоговая таблица должна быть **последним** блоком.

5. **Вступление** — при желании убрать явную отсылку к названию серии курса, оставить тему единицы и страницы (п. 1.1 essence-generation).

### `lesson_voice_11/voice_lesson_11.md`

1. **Шапка** — крошки и «Быстрые ссылки» с `content_11.md` ([voice-lesson-from-digitized.md](../../../docs/voice-lesson-from-digitized.md)).

2. **Ситуация 5** — исправить пустую строку чек-листа; добавить греческие фразы с переводом (например, из блока микрофона в главе: свой / съём, где дом, транспорт, квадратные метры).

3. **Блок 2** — подраздел «Πώς το λένε»: у каждой фразы пара **греч. — рус.** по [voice_lesson_generation.md](../../../docs/promts/voice_lesson_generation.md).

4. **Чек-лист ситуации 1** — убрать формулировку «в оцифровке» у заголовка объявления.
