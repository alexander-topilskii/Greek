## `essence_14/essence_14.md` — контент и структура

### Критично для учебника

- **Противоречие с диалогом главы:** в блоке «мини-диалог в таверне» конспекта эталон — «будем пить белое [вино]», тогда как в оцифровке (диалог B13) компания заказывает **красное** и **килограмм** (ответ на вопрос официанта о вине). Для самопроверки и голосовой отработки привести в соответствие с сюжетом или явно развести «лексический пример» и «сюжет диалога».

### Недостаточный охват темы урока (по сравнению с главой 184–199)

Конспект задаёт только узкий каркас (**θα** + модель **αγοράζω**, несколько глаголов, обрывки фраз). Для **повторения и отработки** без скана не хватает опор из учебника:

- **Календарь и опоры времени** (блок «Για δες» на стр. 187 и аналоги): сегодня / завтра / послезавтра, «в следующем году», «через неделю / месяц», пара **άλλη / επόμενη** (в учебнике на схеме) — в главе это отдельная коммуникативная ось рядом с будущим.
- **Расширенные таблицы будущего:** опорные строки учебника (в т.ч. «быть / иметь / делать / знать / идти / ждать» в парадигме наст./аор./буд.); парадигма **τρώω** с **θα φάω** и т.д.; длинные списки глаголов из цветных рамок (в т.ч. **βάζω, παίρνω, παραγγέλνω, έρχομαι, γίνομαι, βλέπω, λέω, πίνω** …) — без сжатых таблиц в конспекте сложно закрепить материал так, как выстроен урок в книге.
- **Будущее + клитика (местоимение):** примеры вроде *θα την αγοράσουν*, *θα μας φέρετε*, *θα σας πω*, отрицание *Δε θα την κλείσουν* — в оцифровке отдельный блок; в конспекте отсутствует, хотя это центральная связка для «планов и заказа».
- **Тип B1–B2** (*μιλάω* → *θα μιλήσω*, суффиксы *-ήσω / -άσω / -έσω*, длинные списки глаголов на *-άω / -ώ* из учебника) — большой фрагмент главы (стр. 192–193); без него конспект не отражает полноту грамматики единицы.
- **Ключевой диалог B13:** сейчас только укороченный «мини-диалог»; для **самодостаточности** по essence-generation лучше дать **полный или осмысленно сжатый** диалог (как опорный текст для чтения и ролей), а не только обрывки заказа.

### Словарь и практика

- Нет финального раздела **словарь** с **итоговой таблицей лексики** (все уникальные слова и устойчивые формы урока, **включая заголовки и формулировки заданий**, столбцы греч. | рус.) — это обязательный ориентир для конспекта в репозитории и для промпта голосовой отработки по `essence`.
- Разделы оформлены как «1) … 7)» вместо рекомендуемой цепочки **цели → грамматика → фразы/диалоги → эталоны → словарь**; нет **хлебных крошек** и **таблицы быстрых ссылок** (минимум на `content_14.md`), как в AGENTS.md и essence-generation.
- Практика: есть только отсылка к вопросам «где будешь завтра…»; нет **сжатых эталонов** к типовым цепочкам из упражнений (сопоставление продолжений, цепочки *θα … και θα …*, ролевые сценарии) — по желанию, без превращения в полный ключ.

### Язык и словарь в блоке 3

- В русских пояснениях встречается форма вроде **«орекитики / орекитиков»** — искажение от *ορεκτικά*; для ученика лучше **«закуски»**, **«из закусок»** или оставить греч. термин с переводом.
- Проверить строки таблиц на правило промпта: у **существительных** в словаре — **ο / η / το** (где в строке именно существительное, а не целая фраза — выровнять формат).

### Содержание

## Согласованность материалов

- **Голосовой урок** опирается на полные диалоги B13/B14 и ролевые задания; **конспект** сейчас **значительно уже** по грамматике и лексике. Для связки «конспект → голос» стоит расширить `essence_14.md` до уровня охвата главы (хотя бы сжатые таблицы и итоговая лексика), иначе читатель с одним только конспектом не получает той же базы, что заложена в учебнике.
