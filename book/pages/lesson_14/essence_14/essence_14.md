## 📌 Тема и цели

- Строить **будущее**: **Θα σας πω κάτι.** **Θα έρθει τον άλλο μήνα.** **Τι θα πάρετε;**
- Понимать **опоры времени**: **αύριο**, **μεθαύριο**, **σε μια εβδομάδα**, **τον άλλο / επόμενο μήνα**, **του χρόνου**, **την άλλη / επόμενη εβδομάδα**.
- Сопоставлять **τώρα · χτες · αύριο (θα)** в опорных таблицах, включая **τρώω** → **θα φάω** и «зелёный» ряд (**θα δω**, **θα πω** …).
- Употреблять **клитику** после **θα**: **θα την αγοράσουν**, **θα μας φέρετε**, **θα σας πω**, **Δε θα την κλείσουν**.
- Спрягать **тип Β1/Β2**: **θα μιλήσω**, **θα γελάσω**, **θα μπορέσω** и списки на **-ήσω / -άσω / -έσω**.
- Вести **диалог заказа**: ορεκτικά, основное блюдо, **κρασί** (**κόκκινο** / **λευκό**, **χύμα**, **ένα κιλό**).

## 🧠 Грамматика и правила

### Модель **αγοράζω**

| τώρα | χτες | αύριο |
|------|------|-------|
| αγοράζω | αγόρασα | **θα αγοράσω** |
| | | θα αγοράσεις · θα αγοράσει · θα αγοράσουμε · θα αγοράσετε · θα αγοράσουν(ε) |

*Те же окончания, что у настоящего **-ω**, с основой аориста (**αγοράσ-**).*

### Одна основа — настоящее / аорист / **θα**

| παρόν | αορίστος | μέλλοντας (**θα**) |
|-------|----------|---------------------|
| κλείνω | έκλεισα | **θα κλείσω** |
| αλλάζω | άλλαξα | **θα αλλάξω** |
| ανοίγω | άνοιξα | **θα ανοίξω** |
| δουλεύω | δούλεψα | **θα δουλέψω** |
| γράφω | έγραψα | **θα γράψω** |

**Группы основ** (как в уроке 12): **-νω/-ζω → -σω** · **-ζω/-γω/… → -ξω** · **-εύω/-πω/… → -ψω**.

*Дополнительно:* αλλάζω, φωνάζω, παίζω, πειράζω … — та же логика **θα** + основа аориста.

### «Особые» формы (опора)

**θα πω** (**λέω** / **είπα**) · **θα έρθω** (**ήρθα**) · **θα πάω** (**πήγα**) · **θα φάω** (**έφαγα**, от **τρώω**).

### Календарь и опоры (**σήμερα → αύριο / μετά**)

| Σήμερα | Αύριο / μετά |
|--------|----------------|
| Σήμερα είναι Τετάρτη. | Αύριο θα είναι Πέμπτη. Μεθαύριο θα είναι Παρασκευή. |
| Σήμερα είναι Τρίτη 26 Ιουνίου. | *Μετά από* / *Ύστερα από* / *Σε* — μια εβδομάδα θα είναι Τρίτη 3 Ιουλίου. |
| Σήμερα είναι Σάββατο 8 Σεπτεμβρίου. | Σε έναν μήνα θα είναι Δευτέρα 8 Οκτωβρίου. |
| Φέτος είναι 2010. | Του χρόνου θα είναι 2011. |
| Αυτή την εβδομάδα κάνει ζέστη. | Την **άλλη** / **επόμενη** εβδομάδα θα κάνει κρύο. |
| Αυτό τον μήνα διαβάζω για το τεστ. | Τον **άλλο** / **επόμενο** μήνα θα διαβάσω το βιβλίο που μου έδωσες. |

### **Μέλλοντας + αντωνυμίες** (полная форма → клитика)

| Πλήρης μορφή | Σύντομη μορφή |
|---------------|----------------|
| θα αγοράσουν την ταβέρνα. | **θα την αγοράσουν.** |
| θα φέρετε σε εμάς ένα σαγανάκι; | **θα μας φέρετε** ένα σαγανάκι; |
| θα πω σε εσάς κάτι. | **θα σας πω** κάτι. |
| — θα την κλείσουν την ταβέρνα. | — Όχι. **Δε θα την κλείσουν.** |

*Клитика стоит **после** частицы **θα** и **перед** глаголом.*

### Тип **Β1 – Β2**

#### **-ήσω** *(от μιλάω / μιλώ)*

| τώρα | χτες |
|------|------|
| μιλάω / -ώ | μίλησα |

**θα μιλήσω**, θα μιλήσεις … θα μιλήσουν(ε)

**На -ήσω:** ρωτάω → θα ρωτήσω · απαντάω → θα απαντήσω · ξυπνάω → θα ξυπνήσω · συναντάω → θα συναντήσω · περπατάω → θα περπατήσω · βοηθάω → θα βοηθήσω · σταματάω → θα σταματήσω · αργώ → θα αργήσω · τηλεφωνώ → θα τηλεφωνήσω · χρησιμοποιώ → θα χρησιμοποιήσω · ξεναγώ → θα ξεναγήσω · παρακολουθώ → θα παρακολουθήσω · ζω → θα ζήσω

#### **-άσω** *(от γελάω)*

γελάω → γέλασα → **θα γελάσω** …

**На -άσω:** διψάω → θα διψάσω · πεινάω → θα πεινάσω · χαλάω → θα χαλάσω · περνάω → θα περάσω · ξεχνάω → θα ξεχάσω

#### **-έσω** *(от μπορώ)*

μπορώ → μπόρεσα → **θα μπορέσω** …

**На -έσω:** φοράω → θα φορέσω · πονάω → θα πονέσω · καλώ → θα καλέσω

**Παράδειγμα:** Τον Αύγουστο **θα καλέσω** τους φίλους μου από την Ελλάδα στη χώρα μου. **Θα** τους **ξεναγήσω** στην πόλη μου και πιστεύω ότι **θα περάσουμε** πολύ ωραία.

## 💬 Готовые фразы и диалоги

### Вводная сетка

- **Θα σας πω κάτι.** · **Τι θα πάρετε;** · **Τι θα πιούμε;** · **Θα έρθει τον άλλο μήνα.**

### «Πώς το λένε» — после первого диалога

- **Κίτρινο / πορτοκαλί:** Γιατί δεν ερχόμαστε πιο συχνά; · Έλα ντε;
- **Πράσινο:** Παιδιά, παραγγέλνουμε; Έρχεστε, σας παρακαλώ;
- **Μπλε:** Τι θα πάρουμε;
- **Μωβ:** Από ορεκτικά θα μας φέρετε… Θα θέλαμε μία πατάτες…
- **Πορτοκαλί:** Τι θα πιούμε;

### Диалог **«Θα σας πω κάτι!»** (таверна)

**Ερβίν:** Ωραία η ταβέρνα του Θωμά. Τελευταία φορά φάγαμε εδώ πέρσι το φθινόπωρο. Γιατί δεν ερχόμαστε πιο συχνά;  
**Μαρίνα:** Έλα ντε; Μ’ αρέσει αυτή η ταβέρνα! Μ’ αρέσει η αυλή της, τα μεγάλα βαρέλια γύρω γύρω, το σπιτικό φαγητό του Θωμά…  
**Αρλέτα:** …οι γατούλες που ανεβαίνουν στις καρέκλες…  
**Μαρίνα:** Κι αυτές, κι αυτές. Σκέφτομαι ότι όλα αυτά θα αλλάξουν μια μέρα. Ο Θωμάς θα πάρει σύνταξη και θα κλείσει την ταβέρνα. Στη θέση της θα φτιάξουν μια πολυκατοικία ή ένα εμπορικό κέντρο.  
**Ερβίν:** Ε, δε νομίζω. Εγώ λέω ότι κάποιος θα την αγοράσει και θα τη φτιάξει ακόμα πιο ωραία.  
**Βέρα:** Λοιπόν, παιδιά, παραγγέλνουμε; Έρχεστε, σας παρακαλώ;  
**Σερβιτόρος:** Είστε έτοιμοι;  
**Πάμπλο:** Περίπου. Τι θα πάρουμε από ορεκτικά;  
**Μαρίνα:** Θα μας φέρετε ένα σαγανάκι, μία πατάτες…  
**Ερβίν:** Κολοκυθάκια τηγανητά, μία χωριάτικη… Θα φάμε κρέας ή ψάρι;  
**Βέρα:** Προτιμώ κρέας.  
**Ερβίν:** Λοιπόν, θα πάρουμε παϊδάκια για πέντε άτομα;  
**Αρλέτα:** Μια χαρά! Και τι θα πιούμε; Κρασί;  
**Πάμπλο:** Βέβαια!  
**Σερβιτόρος:** Λευκό ή κόκκινο;  
**Όλοι μαζί:** Κόκκινο. Ένα κιλό.  
**Σερβιτόρος:** Ευχαριστώ. Φέρνω το κρασάκι αμέσως.  
**Πάμπλο:** Λοιπόν, Βέρα, τι γίνεται;  
**Βέρα:** Παιδιά, θα σας πω κάτι. Είμαι πολύ χαρούμενη! Το άλλο Σάββατο θα έρθει ο Αλεξέι για έξι μήνες στην Αθήνα!  
**Ερβίν:** Ο Αλεξέι, ο γιος σου; Πώς;  
**Βέρα:** Μ’ ένα πρόγραμμα του Πανεπιστημίου. Θα παρακολουθήσει κάποια σεμινάρια στατιστικής.

*В сюжете компания заказывает **κόκκινο** κρασί **ένα κιλό** (разливное). Утверждение в задании «верно/неверно» про **λευκό** вино относится к **неверным**.*

### Диалог — продолжение вечера

**Ερβίν:** Άντε, Βέρα, με το καλό! Στην υγειά μας, παιδιά!  
**Μαρίνα:** Πόσο είπες ότι θα μείνει ο Αλεξέι;  
**Βέρα:** Έξι μήνες. Θα φύγει τέλος Ιουνίου, αρχές Ιουλίου. Δεν ξέρω ακριβώς. Μάλλον θα έχει εξετάσεις. Τι να σας πω;  
**Πάμπλο:** Τα μαθήματα στο Πανεπιστήμιο θα είναι στα ελληνικά;  
**Βέρα:** Όχι, βρε Πάμπλο. Το πρόγραμμα είναι στα αγγλικά, αλλά θα αρχίσει μαθήματα ελληνικών.  
**Αρλέτα:** Και τι σχέδια έχετε; Θα πάτε κανένα ταξιδάκι έξω από την Αθήνα;  
**Βέρα:** Ε, θα δούμε. Έτσι λέω.  
**Πάμπλο:** Λοιπόν, θα φτιάξω εγώ πρόγραμμα για τον Αλεξέι. Η Μελέκ, ως αρχαιολόγος, θα τον πάει σε μουσεία, αρχαιολογικούς χώρους, θέατρα. Θα μάθει τα πάντα το παιδί.  
**Ερβίν:** Αχ, Αλεξέι, τι σε περιμένει!  
**Πάμπλο:** Εσύ, Μαρίνα, θα οργανώσεις εκδρομές στην Αίγινα, στο Ναύπλιο… Κι εγώ θα τον ξεναγήσω στη νυχτερινή Αθήνα. Ξέρεις, κλαμπ, μπαρ, μπουζούκια…  
**Όλοι μαζί:** Αχ, Αλεξέι! Τι σε περιμένει!

### «Πώς το λένε» — после второго диалога

- **Πράσινο:** Άντε με το καλό! · Στην υγειά μας! · Θα φύγει τέλος Ιουνίου, αρχές Ιουλίου.
- **Κίτρινο:** Δεν ξέρω ακριβώς. Τι να σας πω; · Τι σχέδια έχετε; · Θα δούμε. Έτσι λέω.
- **Μπλε:** Θα οργανώσω εκδρομές. · Θα τον ξεναγήσω στην Αθήνα. · Αχ, τι σε περιμένει!
