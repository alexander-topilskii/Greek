Проверить
θα + форма глагола; αύριο, του χρόνου, την επόμενη εβδομάδα будущее: θα + глагол; «завтра», «в следующем году», «на след. неделе»
τώρα / χτες / θα (таблицы) наст. / прош. / буд.: θα φάω, θα δω, θα έρθω
θα μας φέρετε, θα την αγοράσουν, δε θα την κλείσουν примеры: «принесёте», «купят», «не закроют»
B1 / B2: θα μιλήσω, θα περάσουμε, θα καλέσω будущее типов B1/B2
ορεκτικά, παϊδάκια, κόκκινο / λευκό, ένα κιλό заказ: закуски, ребрышки, вино, кило
