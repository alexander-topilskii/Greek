## Контент и учебная ценность (в приоритете)

### Конспект `essence_15/essence_15.md`

**Файла нет** — для урока-повторения это главный пробел: в [lesson_digitized/lesson_15_digitized.md](lesson_digitized/lesson_15_digitized.md) большой объём **обзорной грамматики** (таблицы времён по моделям Α и В1+В2 из раздела повторения, аномальные и частые глаголы, склонение существительных по трём падежам, личные местоимения с глаголами «видеть» и «говорить»), затем упражнения на порядок предложений, будущее время, предлоги места, местоимённые дополнения, аудирование, переписка, чтение. Без конспекта читатель не получает самодостаточного сценария повторения по [essence-generation.md](../../../docs/essence-generation.md).

**Что стоит заложить в конспект (по смыслу главы)**

- Компактные **опорные таблицы** грамматики из начала главы (не отсылкой к файлам сырья, а **встроенным текстом**).
- **Предлоги места** из цветных блоков — с примерами как в оцифровке.
- **Эталоны** для ключевых упражнений: упорядочивание биографии (задание 1), анкета **Ειρήνη** (2), диалог инструктора с учеником про парковку с формами **θα** + глагол (3), коты и предлоги (4), местоимения в ответах (5) — хотя бы в сжатом виде, где без картинки смысл ясен.
- Задания **6–7 (B16, B17)** завязаны на **аудио**; в конспекте либо кратко указать, что блок только при наличии звука, либо при возможности добавить **транскрипт** диалогов внутрь файла (без ссылки на внешний трек).
- **Упражнение 8** (сообщения α–η и ответы): в оцифровке есть расхождение формулировок (**«14 ответов»** в условии vs фактически показанные номера **1–12** на стр. 207) — при выносе эталонов сверить с печатным пособием и не воспроизводить путаницу.
- Статья про **Мύκονος** (9) и задания **верно/неверно** — краткие эталоны или комментарии к типичным ловушкам (например, остров **Δήλος** географически не «на Миконосе»).
- Финальный раздел словаря (📇 в заголовке, см. [AGENTS.md](../../../AGENTS.md)) и **итоговой таблицей** всей лексики урока (включая заголовки и формулировки заданий).

### `lesson_voice_15/voice_lesson_15.md`

**Что уже хорошо**

- Охвачены **предлоги места**, сцена **парковки**, **коты** + зачатки работы с **τον**, фрагменты **переписки** и **Миконос**, плюс **биографические** фразы и **ролевые** заготовки из конца главы.
- Три блока по структуре промпта в целом соблюдены.

**Пробелы для разговорной отработки**

- Почти не использован **обзор глаголов и существительных** из первых страниц главы — для повторительного урока **«Πάμε πάλι!»** это ядро; в голосовом материале полезны короткие **микродриллы** (например, тройки настоящее — аорист — будущее по нескольким глаголам из таблицы или готовые мини-реплики с **θα**).
- **Местоимения в ответах** (упр. 5): в чек-листе есть только пример с **τον**; нет опор для остальных пунктов — слабая **отработка** по сравнению с главой.
- **Аудиозадания (B16, B17)** не переведены в сценарий без транскрипта — для самостоятельной работы ограничение стоит явно учитывать при доработке.
- **Ситуация 4:** в чек-листе не все **восемь сообщений** и не размечены связки с вариантами ответов — для «ответь на письмо» материал неполный.
- **Ситуация 5:** длинные абзацы статьи о **Миконосе** фактически **скопированы** из оцифровки; по духу [voice_lesson_generation.md](../../../docs/promts/voice_lesson_generation.md) лучше **сжать и перефразировать**, оставив ключевые фразы для устной работы.
- **Ситуация 1:** роли **А/Б** сведены к «по очереди произносите» — слабее, чем **пошаговые инструкции** из промпта; хуже для самостоятельной отработки.
- В чек-листе с **котами** (подписи к рисунку с именами котов) фразы на греческом даны **без русского перевода** в едином формате **греч. — рус.** (требование промпта).

Шапка по [essence-generation.md](../../../docs/essence-generation.md): заголовок с диапазоном **стр. 200–211**, хлебные крошки, узкая таблица **«Быстрые ссылки»** только на `content_15.md` и при необходимости соседние конспекты — **без** ссылок на `digitized/`, `lesson_digitized/`, `voice_lesson_*.md`, `docs/`.

### `lesson_voice_15/voice_lesson_15.md`

1. **Шапка:** хлебные крошки и таблица **«Быстрые ссылки»** с `content_15.md` ([voice-lesson-from-digitized.md](../../../docs/voice-lesson-from-digitized.md)); сейчас файл сразу начинается с блока 1.
2. **Автономность:** убрать формулировку «**(как в учебнике)**» у роли Б в ситуации 4 — это отсылка к изданию; заменить нейтрально («по смыслу» / «выберите ответ из списка **Απαντήσεις**» без слова «учебник»).
3. **Опечатка смешения алфавитов:** в блоке 3 таблицы в ячейке после «πίσω από την» стоят **кириллические** символы (**U+0442**, **U+0430**) вместо греческих **τ** и **α** — должно быть **πόρτα** (см. оцифровку).
4. **Блок 2:** разметка Markdown с **вложенными `**`** внутри жирного (например, у фразы про περίπτερο) ломает единообразие; лучше одна пара звёздочек на всю греческую фразу или курсив только для предлога.
5. После правок при необходимости пересобрать `voice_lesson_15.html` тем же пайплайном, что и для других уроков.
