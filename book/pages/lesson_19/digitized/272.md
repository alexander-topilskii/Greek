## Για δες *(εικονίδιο: μεγεθυντικός φακός)*

*(θηλυκά ουσιαστικά που στον πληθυντικό παίρνουν καταλήξεις **-εις** / **-εων** — όπως στο βιβλίο.)*

### Ενικός αριθμός

| Παραδείγματα | η λέξη | κατάληξη |
|----------------|--------|----------|
| Η αίτηση είναι έτοιμη. | η λέξη | -η |
| Ποιος είναι ο δάσκαλος αυτής της τάξης; | της λέξης | -ης |
| Θα ήθελα να κάνω μια ανάληψη. | τη λέξη | -η |

### Πληθυντικός αριθμός

| Παραδείγματα | οι λέξεις | κατάληξη |
|----------------|-----------|----------|
| Οι δηλώσεις είναι στο γραφείο. | οι λέξεις | -εις |
| Δεν καταλαβαίνω το νόημα αυτών των λέξεων. | των λέξεων | -εων |
| Φοβάμαι τις επισκέψεις στους γιατρούς. | τις λέξεις | -εις |

### Λίστα λέξεων *(μπλε πίνακας)*

| **-ση** | **-ξη** | **-ψη** |
|---------|---------|--------|
| η αίτηση | η απόδειξη | η ανάληψη |
| η απάντηση | η λέξη | η επίσκεψη |
| η άσκηση | η λήξη | η σκέψη |
| η γέννηση | η όρεξη | |
| η δήλωση | η πράξη | |

| η διεύθυνση | η τάξη | |
| η είδηση | | |
| η ειδοποίηση | | |
| η έκδοση | | |
| η ερώτηση | | |
| η θέση | | |
| η κατάθεση | | |

*(Χωριστό πλαίσιο κάτω δεξιά στον πίνακα:)* **η δύναμη**, **η πόλη**.

### Παράδειγμα κλίσης *(κίτρινα «μπλόκ», εικονίδιο: σφραγίδα, λουκέτα)*

## Εικόνα *(κάτω: γυαλιά)*

*(ζευγάρι γυαλιών· σε κάθε φακό λίστες κλίσης.)*

**Αριστερός φακός** — συνηθισμένο θηλυκό σε **-η**:

- η υπογραφή
- οι υπογραφές
- των υπογραφών
- τις υπογραφές

**Δεξιός φακός** — η μελέτη της σελίδας (**-η** / **-εις**):

- η αίτηση
- οι αιτήσεις
- των αιτήσεων
- τις αιτήσεις

*(μπαλόνι: «Αυτές οι αιτήσεις δεν έχουν υπογραφές.»)*
