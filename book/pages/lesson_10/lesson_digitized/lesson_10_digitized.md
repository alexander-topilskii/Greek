## Πάμε πάλι!

### Αριθμοί 1–4 με ουσιαστικά (χυμός, σοκολάτα, παγωτό)

*Το **δύο** μένει ίδιο.*

#### Ονομαστική

| ♂ χυμός | ♀ σοκολάτα | ⚲ παγωτό |
|---------|------------|----------|
| ένας χυμός | μία σοκολάτα | ένα παγωτό |
| δύο χυμοί | δύο σοκολάτες | δύο παγωτά |
| τρεις χυμοί | τρεις σοκολάτες | τρία παγωτά |
| τέσσερις χυμοί | τέσσερις σοκολάτες | τέσσερα παγωτά |

#### Αιτιατική (με «θέλω»)

- **♂** Θέλω **έναν** χυμό. / Θέλω **τρεις** χυμούς. / Θέλω **τέσσερις** χυμούς.
- **♀** Θέλω **μία** σοκολάτα. / Θέλω **τρεις** σοκολάτες. / Θέλω **τέσσερις** σοκολάτες.
- **⚲** Θέλω **ένα** παγωτό. / Θέλω **τρία** παγωτά. / Θέλω **τέσσερα** παγωτά.

---

### Επίθετο σε **-ος, -η, -ο** — *ακριβός*

**Υπολογιστής** (♂), **μηχανή** (♀), **κινητό** (⚲)

| ♂ | ♀ | ⚲ |
| :--- | :--- | :--- |
| Αυτός ο υπολογιστής είναι **ακριβός**. | Αυτή η μηχανή είναι **ακριβή**. | Αυτό το κινητό είναι **ακριβό**. |
| Θέλετε τον **ακριβό** υπολογιστή; | Θέλετε την **ακριβή** μηχανή; | Θέλετε το **ακριβό** κινητό; |
| Αυτοί οι υπολογιστές είναι **ακριβοί**. | Αυτές οι μηχανές είναι **ακριβές**. | Αυτά τα κινητά είναι **ακριβά**. |
| Θέλετε τους **ακριβούς** υπολογιστές; | Θέλετε τις **ακριβές** μηχανές; | Θέλετε τα **ακριβά** κινητά; |

---

### Επίθετο σε **-ος, -α, -ο** — *καινούριος*

| ♂ | ♀ | ⚲ |
| :--- | :--- | :--- |
| Αυτός ο υπολογιστής είναι **καινούριος**. | Αυτή η μηχανή είναι **καινούρια**. | Αυτό το κινητό είναι **καινούριο**. |
| Θέλω τον **καινούριο** υπολογιστή. | Θέλω την **καινούρια** μηχανή. | Θέλω το **καινούριο** κινητό. |
| Αυτοί οι υπολογιστές είναι **καινούριοι**. | Αυτές οι μηχανές είναι **καινούριες**. | Αυτά τα κινητά είναι **καινούρια**. |
| Θέλουμε τους **καινούριους** υπολογιστές. | Θέλουμε τις **καινούριες** μηχανές. | Θέλουμε τα **καινούρια** κινητά. |

---

### Αντωνυμίες γενικής (για ποιον; για ποια;)

| | |
| :--- | :--- |
| **Μου** δίνετε μια ταυτότητα; | |
| **Σου** μιλάω, αλλά δεν ακούς! | |
| **Του** αγοράζω εφημερίδα… *(στον Νίκο ♂)* | |
| **Της** στέλνω e-mail συχνά. *(στην Ελένη ♀)* | |

| **Σας** διαβάζω το γράμμα. | |
| **Τους** λέμε τα νέα μας. *(στους φίλους ♂)* | |
| **Τους** στέλνετε κάρτες; *(στις φίλες ♀)* | |

---

### Το ρήμα *αρέσει* / *αρέσουν*

**Με μου, σου, του, της, του, μας, σας, τους:**

- **Ενικό θέμα** → **αρέσει**  
  … μου αρέσει **ο** κινηματογράφος · … της αρέσει **η** θάλασσα · … του αρέσει **το** βιβλίο

- **Πληθυντικό θέμα** → **αρέσουν**  
  … μας αρέσουν **οι** χοροί · … τους αρέσουν **οι** τυρόπιτες · … τους αρέσουν **τα** σουβλάκια

### 5. Λεωφορεία, μετρό, τραμ, τρόλεϊ… *(A59)*

Θα ακούσω δύο φορές έναν διάλογο στον δρόμο. Ένας κύριος κάνει μια έρευνα για τα μέσα μεταφοράς (λεωφορεία, τρόλεϊ, τραμ, μετρό…) στην πόλη του. Συμπληρώνω τις απαντήσεις που παίρνει, όπως στο παράδειγμα.

1. Ο κύριος κάνει μια έρευνα για **το Πανεπιστήμιο.** *(χειρόγραφο στο βιβλίο)*

2. Η κυρία μένει ________________________________.

3. Η κυρία δε μένει κοντά στον ________________________________.

4. Στη δουλειά της πηγαίνει με:

   α. ________________________________  
   β. ________________________________  
   γ. ________________________________  
   δ. ________________________________  

5. Όταν το λεωφορείο αργεί, περιμένει στη στάση ________________________________.

6. Το πρωί κάνει συνήθως ________________________________ μέχρι τη δουλειά της και το απόγευμα ________________________________.

7. Όταν είναι κουρασμένη, καμιά φορά ________________________________.

8. Το λεωφορείο μέχρι τον σταθμό του μετρό κάνει ________________________________.

9. Το μετρό μέχρι την Ομόνοια κάνει ________________________________.

10. Ο ηλεκτρικός από την Ομόνοια μέχρι τα Κάτω Πατήσια κάνει ________________________________.

11. Η κυρία έχει αυτοκίνητο, αλλά ________________________________.

Η Λένα και ο Νίκος μιλάνε για τον ελεύθερο χρόνο τους, το διάβασμα, την τηλεόραση… Ακούω τον διάλογο δύο φορές και βάζω ✓ στο σωστό ή στο λάθος.

| # | Πρόταση | Σωστό | Λάθος |
|---|---------|:-----:|:-----:|
| 1 | Η Λένα δουλεύει σε ένα τουριστικό γραφείο. | ✓ | |
| 2 | Η Λένα εδώ και δύο χρόνια ταξιδεύει πολύ. | | |
| 3 | Ο Νίκος φεύγει κάθε μέρα στις δέκα το βράδυ από το γραφείο. | | |
| 4 | Όταν ο Νίκος γυρίζει αργά, βλέπει τηλεόραση και πάει για ύπνο. | | |
| 5 | Η Λένα δε βλέπει τηλεόραση ποτέ. | | |
| 6 | Η Λένα συνήθως μαθαίνει τα νέα από το Ίντερνετ. | | |
| 7 | Και ο Νίκος και η Λένα ξέρουν μουσική. | | |
| 8 | Η Λένα παίζει πιάνο κάθε μέρα. | | |
| 9 | Ο Νίκος βγαίνει μόνο το Σαββατοκύριακο. | | |

## Παίζω έναν ρόλο *(μάσκες)*

### 1.1 Στο ταξί

**Ρόλος Α**

Μπαίνω σε ένα ταξί. Ο / η οδηγός με ρωτάει πολλά πράγματα για τη ζωή μου. Τον / την ρωτάω και εγώ για την οικογένειά του, τη δουλειά του και ό,τι άλλο θέλω.

**Ρόλος Β**

Είμαι οδηγός ταξί. Ένας κύριος / μια κυρία μπαίνει στο ταξί μου. Καταλαβαίνω ότι δεν είναι από την Ελλάδα και τον / την ρωτάω από πού είναι, για τη χώρα του / της και ό,τι άλλο θέλω.

---

### 1.2 Στον μανάβη

**Ρόλος Α**

Είμαι στο μανάβικο και αγοράζω φρούτα και λαχανικά για το τραπέζι που έχω το Σάββατο το βράδυ. Θέλω πολλά πράγματα, αλλά είναι ακριβά.

**Ρόλος Β**

Δουλεύω στο μανάβικο. Ο πελάτης / η πελάτισσα ζητάει πολλά πράγματα. Μερικά από αυτά νομίζει ότι είναι ακριβά. Απαντώ στις ερωτήσεις και του / της δίνω αυτά που θέλει.
