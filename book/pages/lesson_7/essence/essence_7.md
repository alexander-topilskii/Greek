# 🔹 2. Распорядок дня

## 📌 Базовая модель

- Το πρωί ξυπνάω νωρίς
- Πάω στη δουλειά
- Το μεσημέρι τρώω
- Το απόγευμα βλέπω τηλεόραση
- Το βράδυ βγαίνω έξω
- Πάω για ύπνο

## 📌 Временные маркеры

| Греческий   | Перевод               |
|-------------|-----------------------|
| το πρωί     | утром                 |
| το μεσημέρι | днём                  |
| το απόγευμα | после обеда / вечером |
| το βράδυ    | вечером / ночью       |

### ⚠️ Пояснение

В греческом **время дня ставится в начале** и не требует предлога:

- ❌ «утром я…»
- ✅ Το πρωί + глагол

---

# 🔹 3. Частотность

## 📌 Наречия

| Греческий     | Перевод           |
|---------------|-------------------|
| πάντα         | всегда            |
| συνήθως       | обычно            |
| συχνά         | часто             |
| πολλές φορές  | много раз / часто |
| μερικές φορές | иногда            |
| καμιά φορά    | иногда (реже)     |
| κάπου κάπου   | время от времени  |
| σπάνια        | редко             |
| ποτέ          | никогда           |
| σχεδόν ποτέ   | почти никогда     |

### ⚠️ Пояснение (очень важно)

- **Ποτέ = никогда**, если есть отрицание:
    - Ποτέ δεν δουλεύω → Я никогда не работаю

- Без отрицания может значить «когда-нибудь» (реже в A1)

- Порядок:
    - наречие + глагол
    - Συχνά βλέπω τηλεόραση

# 🔹 4. Глаголы

## 📍 λέω — говорить

| Лицо      | Форма |
|-----------|-------|
| εγώ       | λέω   |
| εσύ       | λες   |
| αυτός/ή/ό | λέει  |
| εμείς     | λέμε  |
| εσείς     | λέτε  |
| αυτοί     | λένε  |

---

## 📍 τρώω — есть

| Лицо  | Форма |
|-------|-------|
| εγώ   | τρώω  |
| εσύ   | τρως  |
| αυτός | τρώει |
| εμείς | τρώμε |
| εσείς | τρώτε |
| αυτοί | τρώνε |

---

## 📍 ακούω — слушать

| Лицо  | Форма  |
|-------|--------|
| εγώ   | ακούω  |
| εσύ   | ακούς  |
| αυτός | ακούει |
| εμείς | ακούμε |
| εσείς | ακούτε |
| αυτοί | ακούνε |

---

### ⚠️ Пояснение

- Это **не полностью регулярные глаголы**
- Важно запомнить изменения:
    - λέω → λες
    - τρώω → τρως
    - ακούω → ακούς

- Во множественном числе:
    - -με / -τε / -νε

# 🔹 5. Вопрос «как долго?»

## 📌 Модель

- Πόσο καιρό είσαι εδώ;
- Πόσο καιρό δουλεύεις;
- Πόσο καιρό κάνεις ελληνικά;

## 📌 Ответы

| Греческий          | Перевод      |
|--------------------|--------------|
| Μόνο πέντε μέρες   | всего 5 дней |
| Μία εβδομάδα       | одна неделя  |
| Έναν μήνα          | один месяц   |
| Δύο μήνες          | два месяца   |
| Έναν χρόνο         | один год     |
| Εδώ και δύο χρόνια | уже 2 года   |
| Πέντε χρόνια τώρα  | уже 5 лет    |
| Από το 2001        | с 2001 года  |

### ⚠️ Пояснение

- **Εδώ και + время** → действие началось в прошлом и продолжается  
  (аналог Present Perfect Continuous)

- **Από + год** → точка начала

# 🔹 6. Дни недели

| Греческий         | Перевод       |
|-------------------|---------------|
| τη Δευτέρα        | в понедельник |
| την Τρίτη         | во вторник    |
| την Τετάρτη       | в среду       |
| την Πέμπτη        | в четверг     |
| την Παρασκευή     | в пятницу     |
| το Σάββατο        | в субботу     |
| την Κυριακή       | в воскресенье |
| το Σαββατοκύριακο | выходные      |

### ⚠️ Пояснение

- Используется **винительный падеж** (кого? что? когда?)
- Артикль обязателен:
    - την Κυριακή
    - το Σάββατο

# 🔹 8. Времена года и погода

| Греческий    | Перевод |
|--------------|---------|
| ο χειμώνας   | зима    |
| η άνοιξη     | весна   |
| το καλοκαίρι | лето    |
| το φθινόπωρο | осень   |

## 📌 Погода

- Τι καιρό κάνει; → Какая погода?
- Κάνει κρύο → холодно
- Κάνει ζέστη → жарко
- Κάνει λίγο κρύο → немного холодно

### ⚠️ Пояснение

- Используется конструкция:
    - **κάνει + состояние**
- Буквально: «делает холод/жару»

# 🔹 9. Дни, время + регулярность

## 📌 Примеры

- Κάθε μέρα πάω στη δουλειά
- Σχεδόν πάντα βγαίνω έξω το Σάββατο
- Συνήθως πίνω καφέ το πρωί
- Συχνά βλέπω τηλεόραση στο σπίτι
- Πολλές φορές έχω μάθημα το απόγευμα
- Μερικές φορές μένω στο σπίτι το βράδυ
- Καμιά φορά πάω σινεμά αργά
- Κάπου κάπου πάω για ψώνια
- Ποτέ πάω για φαγητό έξω (✔ из книги, разговорный вариант)
- Σπάνια πάω για ύπνο νωρίς
- Ποτέ δεν δουλεύω την Κυριακή

### ⚠️ Пояснение

- **Κάθε μέρα** = регулярность
- **наречие + глагол** — стандартный порядок

# 🔹 10. Диалоги и разговорные фразы

## 📌 Общение

- Τι νέα; → Что нового?
- Όλα καλά; → Всё хорошо?
- Πώς πάει; → Как дела?
- Περνάω πολύ καλά → Я отлично провожу время

---

## 📌 Приглашения

- Παρακαλώ…
- Τι λες, πάμε για καφέ;
- Γιατί δεν τα λέμε το βράδυ;
- Έγινε. Τα λέμε το βράδυ.

## 📌 Ответы

- Ναι, γιατί όχι;
- Ναι, αμέ!
- Δεν το πιστεύω!
- Δεν μου λες…
- Κανονίζουμε τίποτα;
- Ωραία ιδέα

# 🔹 13. Ключевые конструкции

## 📌 Частота

# 🔹 15. Важные нюансы урока (итог)

1. Время ставится в начало:  
   → Το πρωί δουλεύω

2. Частотность перед глаголом:  
   → Συχνά τρώω έξω

3. Дни и месяцы = винительный падеж

4. Εδώ και = длительность (до настоящего момента)

5. Ποτέ + δεν = никогда

6. Κάνει + погода
