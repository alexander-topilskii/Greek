# 🎙 Голосовой урок по оцифровке главы

Этот документ задаёт **стандарт создания «голосового урока»**: практический раздаточный материал для разговорной
отработки, **строго по тексту** единой цифровой главы урока. **Вход для генерации** (автор / ИИ) и **итоговый файл для читателя** — разные по правилам ссылок (см. ниже).

---

## Назначение

| Роль | Что это |
|------|---------|
| **Вход для генерации** | Файл **`book/pages/lesson_N/lesson_digitized/lesson_N_digitized.md`** — полный текст главы в Markdown; к нему прикладывают [promts/voice_lesson_generation.md](promts/voice_lesson_generation.md) в сессии ИИ. |
| **Итог для читателя** | **`book/pages/lesson_N/lesson_voice_N/voice_lesson_N.md`** — **автономный** учебный файл: в шапке и теле **нет** markdown-ссылок на `lesson_digitized`, `digitized`, `raw`, на конспект `essence`, на `essence_N.html` / Voice HTML, на другой «остров» материала. Условия и лексика из главы **перенесены в текст** при необходимости, без отсылок к файлам сырья. |
| **Хаб урока** | Ссылка на голосовой урок размещается в **`book/pages/lesson_N/content_N.md`** (таблица «Быстрые ссылки» и т.п.); там же перечисляются сканы, оцифровки, конспект, HTML — это **не** нарушение автономности `voice_lesson_N.md`. |

Голосовая сессия с ИИ по конспекту — отдельный сценарий ([promt/voice_essence_notes_promt.md](promt/voice_essence_notes_promt.md), [promts/ai_voice_promt.md](promts/ai_voice_promt.md), [voice-generation.md](voice-generation.md)). При желании в **одной сессии** можно **прикладывать** файлы `voice_lesson_N.md` и конспект как **вложения к чату** — это не требует ссылок между этими файлами внутри markdown.

---

## Структура каталогов и имя файла

Для урока с номером `N` (пример: `1`, `9`, `20`):

| Элемент                | Путь                                                         |
|------------------------|--------------------------------------------------------------|
| Вход (оцифровка главы) | `book/pages/lesson_N/lesson_digitized/lesson_N_digitized.md` |
| Папка голосового урока | `book/pages/lesson_N/lesson_voice_N/`                        |
| Файл материала         | `book/pages/lesson_N/lesson_voice_N/voice_lesson_N.md`       |

Имя папки **`lesson_voice_N`** должно совпадать с номером урока в родительском каталоге `lesson_N`.

---

## Оформление `voice_lesson_N.md`

Следуй общим правилам репозитория из [agents.md](../agents.md) с **исключением** для хлебных крошек (см. раздел 2 в `agents.md`).

- Заголовок первого уровня `# ...` с уместным эмодзи.
- Сразу после заголовка — **хлебные крошки** с корректным относительным путём к [Readme.md](../Readme.md) и к **`content_N.md`**.
- Таблица **«Быстрые ссылки»** — **только** строки, не нарушающие автономность: как минимум **`content_N.md`**; при необходимости — нейтральные строки без сырья (например внутренние якоря). **Не добавляй** в эту таблицу `lesson_N_digitized.md`, `essence_N.md`, `essence_N.html`, `voice_lesson_N.html`, пути `digitized/` / `raw/`.
- Далее — **только сгенерированный по промпту** из [promts/voice_lesson_generation.md](promts/voice_lesson_generation.md) материал (три блока), без мета-рассуждений в теле файла.

**Запрет на выдумки:** в содержимое блоков нельзя добавлять лексику или грамматику, которых нет в тексте, приложенном к **сессии генерации** (`lesson_N_digitized.md`).

## Ссылка в `content_N.md`

В таблице «Быстрые ссылки» добавь строку (подстрой номер и подпись при необходимости):

| Метка             | Ссылка                                                |
|-------------------|-------------------------------------------------------|
| 🎙 Голосовой урок | [voice_lesson_N.md](lesson_voice_N/voice_lesson_N.md) |

Размещай строку рядом с остальными материалами урока.

## Промпт для генерации (скопировать в ИИ)

Текст промпта вынесен в отдельный файл: **[promts/voice_lesson_generation.md](promts/voice_lesson_generation.md)**. В нём один fenced-блок с меткой `text` — копируют **содержимое** блока (без строк с тройными обратными кавычками) в ИИ вместе с **`lesson_N_digitized.md`**.

---

## Чеклист после генерации

|  Шаг  | Действие                                                                                                                                                                                                                                                                                                                        |
|:-----:|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **1** | **Подготовка файла:** Создать каталог `lesson_voice_N/`, сохранить `voice_lesson_N.md` с шапкой по этому документу и [agents.md](../agents.md).                                                                                                                                                                                 |
| **2** | **Проверка качества генерации (визуальный контроль):**<br>— Текст не обёрнут в глобальные блоки кода ( ``` ).<br>— Нет технического мусора из исходника (HTML-теги, пометки цвета `*(ροζ)*`).<br>— У всех существительных в таблицах словаря проставлены артикли (*ο, η, το*).<br>— Везде присутствует перевод на русский язык. |
| **3** | **Проверка лексики:** Убедиться, что ИИ не придумал лексику или грамматику, которых нет в `lesson_N_digitized.md` (по тексту сессии генерации).                                                                                                                                                                                 |
| **4** | **Автономность файла:** В шапке и теле `voice_lesson_N.md` нет ссылок на сырьё, конспект, Voice HTML, см. раздел «Оформление».                                                                                                                                                                                                  |
| **5** | **Интеграция:** Добавить ссылку на «Голосовой урок» в **`content_N.md`**.                                                                                                                                                                                                                                                       |
| **6** | **Валидация ссылок:** Проверить ссылки из `content_N.md` и допустимые ссылки из шапки `voice_lesson_N.md` (крошки, `content_N.md`).                                                                                                                                                                                             |

## Связь с другими артефактами

Перекрёстные markdown-ссылки **между** `voice_lesson_N.md` и `essence_N/essence_N.md` **не требуются** и **не должны** подменять автономность каждого файла. Навигация читателя — через **`content_N.md`**.

| Артефакт | Роль |
|----------|------|
| `lesson_N_digitized.md` | Обязательный **вход** пайплайна генерации `voice_lesson_N.md`; в итоговом `voice_lesson_N.md` на него **не** ссылаются. |
| `essence_N/essence_N.md` | Конспект по [essence-generation.md](essence-generation.md); обзор слоёв — [lesson-extraction-from-textbook.md](lesson-extraction-from-textbook.md). Не дублировать навигацию на голосовой урок внутри конспекта. |
| [promt/voice_essence_notes_promt.md](promt/voice_essence_notes_promt.md) · [promts/ai_voice_promt.md](promts/ai_voice_promt.md) | Текст промпта для сессии с **конспектом** и методичка; к сессии можно **прикладывать** `voice_lesson_N.md` как второй файл — без ссылки на него из тела конспекта или наоборот. |
| [promts/voice_lesson_generation.md](promts/voice_lesson_generation.md) | Промпт сборки **`voice_lesson_N.md`** из оцифровки главы (три блока: ситуации, шпаргалка, словарь). |
