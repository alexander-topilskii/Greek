Этот документ задаёт **стандарт создания «голосового урока»**: практический раздаточный материал для разговорной отработки, **строго по тексту** единой цифровой главы урока из `lesson_digitized/`.

---

## Назначение

- **Источник истины для генерации** — файл **`book/pages/lesson_N/lesson_digitized/lesson_N_digitized.md`** (полный текст главы в Markdown).
- **Результат** — один Markdown-файл в отдельной папке урока, плюс **явная ссылка** из **`book/pages/lesson_N/content_N.md`** в таблице «Быстрые ссылки».
- Материал дополняет цепочку «страницы → оцифровка → конспект / голос»: при желании его можно приложить к сессии с ИИ наряду с [ai_voice_promt.md](ai_voice_promt.md) и `essence_N.md` (см. раздел «Связь с другими артефактами»).

---

## Структура каталогов и имя файла

Для урока с номером `N` (пример: `1`, `9`, `20`):

| Элемент                | Путь                                                         |
|------------------------|--------------------------------------------------------------|
| Вход (оцифровка)       | `book/pages/lesson_N/lesson_digitized/lesson_N_digitized.md` |
| Папка голосового урока | `book/pages/lesson_N/lesson_voice_N/`                        |
| Файл материала         | `book/pages/lesson_N/lesson_voice_N/voice_lesson_N.md`       |

Имя папки **`lesson_voice_N`** должно совпадать с номером урока в родительском каталоге `lesson_N`.

## Оформление `voice_lesson_N.md`

Следуй общим правилам репозитория из [AGENTS.md](../AGENTS.md):

- Заголовок первого уровня `# ...` с уместным эмодзи.
- Сразу после заголовка — **хлебные крошки** с корректным относительным путём к [Readme.md](../Readme.md) и к **`content_N.md`**.
- Таблица **«Быстрые ссылки»** (минимум: оцифровка `lesson_N_digitized.md`, `content_N.md`, при наличии — `essence_N.md` и строка Voice HTML).
- Далее — **только сгенерированный по промпту ниже материал** (три блока), без мета-рассуждений в теле файла.

**Запрет на выдумки:** в содержимое блоков нельзя добавлять лексику или грамматику, которых нет в приложенном к генерации тексте `lesson_N_digitized.md`.

## Промпт для генерации (скопировать в ИИ)

В чат с моделью вставь **текст промпта** ниже и **приложи** (или вставь целиком) файл `lesson_N_digitized.md` соответствующего урока.

```text
Ты — опытный методист и преподаватель новогреческого языка. Твоя задача — проанализировать предоставленный текст урока и составить на его основе практический раздаточный материал для студентов.

Материал должен быть строго привязан к лексике и грамматике данного текста, ориентирован на развитие разговорных навыков (говорение) и состоять из трех блоков. Используй форматирование Markdown для удобного чтения.

### Блок 1: Коммуникативные ситуации и ролевые игры
Определи главные темы предоставленного урока и создай от 2 до 5 ситуаций для работы в парах на основе диалогов и упражнений. Для каждой ситуации пропиши четкие инструкции для Роли А и Роли Б.
Формат инструкций:
* Опиши контекст (Где находятся собеседники? Какова их цель?).
* Дай пошаговые инструкции для построения диалога (например: «Поздоровайтесь -> Задайте вопрос по теме -> Выразите свое мнение -> ...»).

### Блок 2: Тематический словарь (Таблицы)
Собери лексику из текста урока и разбей ее на 4-6 логических смысловых групп в виде таблиц (Слово на греческом | Перевод на русский в контексте урока).
Правила для словаря:
* Для ВСЕХ существительных обязательно указывай определенный артикль (ο, η, το).
* Глаголы указывай в 1-м лице единственного числа настоящего времени или в форме, в которой они даны в тексте (с пометкой).
* Обязательно выдели в отдельные таблицы: существительные по подтемам, ключевые глаголы, прилагательные/наречия.

### Блок 3: Шпаргалка для диалогов (Фразы-клише)
Выпиши готовые разговорные фразы и устойчивые выражения из текста и сгруппируй их по функции, чтобы студенты могли использовать их как опору во время ролевых игр.
Формат вывода: `* **Греческая фраза** — *Перевод на русский*`
Обязательные базовые группы (адаптируй под текст):
* Вопросы и запрос информации.
* Выражение мнения / согласия / отказа.
* Специфические целевые фразы, характерные исключительно для темы данного урока.

Строго опирайся только на предоставленный текст. Не добавляй новые слова или грамматику, которых нет в материале урока. Выдавай только готовый материал, структурированно, без вводных и заключительных фраз.
```
