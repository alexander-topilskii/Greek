Этот документ задаёт **стандарт создания «голосового урока»**: практический раздаточный материал для разговорной
отработки, **строго по тексту** единой цифровой главы урока из `lesson_digitized/`.

---

## Назначение

- **Источник истины для генерации** — файл **`book/pages/lesson_N/lesson_digitized/lesson_N_digitized.md`** (полный
  текст главы в Markdown).
- **Результат** — один Markdown-файл в отдельной папке урока, плюс **явная ссылка** из *
  *`book/pages/lesson_N/content_N.md`** в таблице «Быстрые ссылки».
- Материал дополняет цепочку «страницы → оцифровка → конспект / голос»: при желании его можно приложить к сессии с ИИ
  наряду с [ai_voice_promt.md](ai_voice_promt.md) и `essence_N.md` (см. раздел «Связь с другими артефактами»).

---

## Структура каталогов и имя файла

Для урока с номером `N` (пример: `1`, `9`, `20`):

| Элемент                | Путь                                                         |
|------------------------|--------------------------------------------------------------|
| Вход (оцифровка)       | `book/pages/lesson_N/lesson_digitized/lesson_N_digitized.md` |
| Папка голосового урока | `book/pages/lesson_N/lesson_voice_N/`                        |
| Файл материала         | `book/pages/lesson_N/lesson_voice_N/voice_lesson_N.md`       |

Имя папки **`lesson_voice_N`** должно совпадать с номером урока в родительском каталоге `lesson_N`.

---

## Оформление `voice_lesson_N.md`

Следуй общим правилам репозитория из [AGENTS.md](../AGENTS.md):

- Заголовок первого уровня `# ...` с уместным эмодзи.
- Сразу после заголовка — **хлебные крошки** с корректным относительным путём к [Readme.md](../Readme.md) и к *
  *`content_N.md`**.
- Таблица **«Быстрые ссылки»** (минимум: оцифровка `lesson_N_digitized.md`, `content_N.md`, при наличии — `essence_N.md`
  и строка Voice HTML).
- Далее — **только сгенерированный по промпту ниже материал** (три блока), без мета-рассуждений в теле файла.

**Запрет на выдумки:** в содержимое блоков нельзя добавлять лексику или грамматику, которых нет в приложенном к
генерации тексте `lesson_N_digitized.md`.

## Промпт для генерации (скопировать в ИИ)

```text
Ты — опытный методист и преподаватель новогреческого языка. Твоя задача — проанализировать предоставленный оцифрованный текст урока и создать чистый, готовый к выдаче ученику раздаточный материал. 

КРИТИЧЕСКИЕ ОГРАНИЧЕНИЯ (СТРОГО СОБЛЮДАТЬ):
1. ЗАПРЕЩЕНО копировать куски исходного текста целиком. Ты должен синтезировать и перерабатывать материал.
2. ЗАПРЕЩЕНО использовать техническую разметку из исходника. Удали все HTML-теги, пометки цвета, иконки и комментарии (например: *(μπλε πλαίσιο)*, *(εικονίδιο)*, *(A49)*, <a id="..."></a>).
3. ОБЯЗАТЕЛЬНЫЙ ПЕРЕВОД: Ни одно греческое слово или фраза в словаре и шпаргалках не должны оставаться без точного перевода на русский язык.
4. Выдавай только готовый материал в Markdown. Никаких приветствий, рассуждений или тегов блоков кода (```) вокруг всего ответа.

СТРУКТУРА МАТЕРИАЛА:

### Блок 1: Коммуникативные ситуации (Ролевые игры)
Выдели 3-5 главных разговорных тем урока. Для каждой создай ситуацию по шаблону:
* **Ситуация [Номер]: [Название ситуации]**
* **Контекст:** [Коротко, где происходит действие и какая цель (1 предложение)]
* **Роль А:** [Пошаговая инструкция на русском, что нужно сказать/спросить]
* **Роль Б:** [Пошаговая инструкция на русском, как реагировать]
* 🎯 **Чек-лист для ситуации:** [3-5 самых важных греческих фраз/слов ИМЕННО для этого диалога из текста урока с переводом на русский. Формат: *Греческий* — Русский].

### Блок 2: Универсальная шпаргалка (Фразы-клише)
Собери полезные разговорные фразы со всего урока и разбей их на 3-4 смысловые группы (например: Вопросы, Согласие/Отказ, Приветствия/Прощания, Специфические фразы темы).
Формат: Маркированный список. Строго: **Греческая фраза** — *Перевод на русский*.

### Блок 3: Тематический словарь
Выпиши новую лексику урока и сгруппируй её в 4-6 таблиц по смыслу (Например: Продукты, Глаголы, Прилагательные, Места).
Формат таблиц: 
| Слово на греческом | Перевод на русский |
Правила: 
- Для ВСЕХ существительных обязательно указывай определенный артикль (ο, η, το).
- Глаголы пиши в 1-м лице ед.ч. (εγώ) или в форме из текста.
```

|  Шаг  | Действие                                                                                                                                                                                                                                                                                                                        |
|:-----:|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **1** | **Подготовка файла:** Создать каталог `lesson_voice_N/`, сохранить `voice_lesson_N.md` с шапкой по [AGENTS.md](../AGENTS.md).                                                                                                                                                                                                   |
| **2** | **Проверка качества генерации (Визуальный контроль):**<br>— Текст не обернут в глобальные блоки кода ( ``` ).<br>— Нет технического мусора из исходника (HTML-теги, пометки цвета `*(ροζ)*`).<br>— У всех существительных в таблицах словаря проставлены артикли (*ο, η, το*).<br>— Везде присутствует перевод на русский язык. |
| **3** | **Проверка лексики:** Вставить текст в файл и убедиться, что ИИ не придумал лексику или грамматику, которых нет в `lesson_N_digitized.md`.                                                                                                                                                                                      |
| **4** | **Интеграция:** Добавить ссылку на «Голосовой урок» в основной навигационный файл `content_N.md`.                                                                                                                                                                                                                               |
| **5** | **Валидация ссылок:** Пройтись по ссылкам из `content_N.md` и из шапки `voice_lesson_N.md`, чтобы убедиться, что нет битых путей.                                                                                                                                                                                               |
