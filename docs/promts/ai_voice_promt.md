# 🎙 Промпт для голосовой отработки (Voice Mode) + конспект

*Инструкция для сессии с ИИ: приложите файл **`book/pages/lesson_N/essence_N/essence_N.md`**
нужного урока. Один и тот же текст промпта подходит для **любого** урока. Для просмотра в браузере без ручной сборки
текста откройте **`essence_N.html`** нужного урока (ссылка в **`content_N.html`** на GitHub Pages или в репозитории).*

---

## Промпт (канонический текст)

**Текст, который копирует кнопка в HTML и читает генератор:** [../promt/voice_essence_notes_promt.md](../promt/voice_essence_notes_promt.md) (внутри файла — один блок в ограждении `text`).

Копировать для чата вручную: откройте этот файл или разверните блок промпта на странице **`essence_N.html`**.

---

## Как пользоваться

| Шаг | Действие                                                                                                                          |
|-----|-----------------------------------------------------------------------------------------------------------------------------------|
| 1   | Откройте **`book/pages/lesson_N/essence_N/essence_N.md`** нужного урока.                                                                    |
| 2   | Скопируйте промпт из [voice_essence_notes_promt.md](../promt/voice_essence_notes_promt.md) (внутри ограждения **text** — без строк с тройными кавычками, если клиент не любит markdown). |
| 3   | Вставьте промпт в голосовой чат и **прикрепите** тот же файл конспекта (или вставьте его содержимое, если вложения нет).     |
| 4   | Уточните при желании уровень или фокус одной фразой после промпта (например: «сегодня только приветствия и είμαι»).               |

*Имя файла `ai_voice_promt.md` — осознанное написание; файл остаётся **методичкой** в `docs/promts/`. Сам текст промпта для встраивания в HTML — в **[docs/promt/voice_essence_notes_promt.md](../promt/voice_essence_notes_promt.md)**.*

---

## Связь с конспектом

- Промпт рассчитан на структуру конспекта из [essence-generation.md](../essence-generation.md):
  грамматика → фразы → словарь (без раздела ключей к заданиям пособия).
- **Диалоги** в `essence_N` отрабатываются **после** микро-блоков по отдельным фразам — это зашито в греческий блок
  «Πρόσθετη προτεραιότητα — διάλογοι».

## Связь с голосовым уроком по оцифровке

- Отдельный пайплайн: раздаточный материал **`voice_lesson_N.md`** (генерация из **`lesson_N_digitized.md`**) — см. [voice-lesson-from-digitized.md](../voice-lesson-from-digitized.md). Его можно **прикладывать к этой же голосовой сессии** вместе с конспектом **`book/pages/lesson_N/essence_N/essence_N.md`** или использовать как основной опорный текст, если конспекта ещё нет (без ссылок между этими файлами в markdown — см. методички).
