# 📖 Правила выноса «урока» из учебника

**[🏠 Readme](../Readme.md) → [🧭 agents.md](../AGENTS.md) → 📄 `docs/lesson-extraction-from-textbook.md`**

*Методичка для подготовки материалов курса по книге «Ελληνικά Α'»: что считать **единицей урока**, как связаны папки,
сканы и конспект, как оформлять **`essence_N.md`**, чтобы им можно было пользоваться **без открытого учебника**.*

| ⚡ Быстрые ссылки         |                                                                                             |
|--------------------------|---------------------------------------------------------------------------------------------|
| 🧩 Формат `.md`          | [AGENTS.md](../AGENTS.md)                                                                   |
| 📚 Точка входа по уроку  | `book/pages/lesson_N/content_N.md`                                                          |
| 💎 Конспект              | `book/pages/lesson_N/essence_N.md`                                                          |
| 🖼 Сканы                 | `book/pages/lesson_N/raw/{номер}.png`                                                       |
| 🔧 Регенерация `content` | [scripts/generate_book_lesson_content_md.py](../scripts/generate_book_lesson_content_md.py) |

## 1. Что такое «урок» в этом репозитории

| Слой                      | Файлы                                   | Назначение                                                                    |
|---------------------------|-----------------------------------------|-------------------------------------------------------------------------------|
| **Скан**                  | `lesson_N/raw/*.png`                    | Источник истины: страницы учебника по номеру в колонтитуле.                   |
| **Оцифровка**             | `lesson_N/digitized/K.md` (опционально) | Дословный или близкий текст со страницы `K` для поиска и цитирования.         |
| **Навигация**             | `lesson_N/content_N.md`                 | Список страниц, превью, ссылки на `raw` и `digitized`; генерируется скриптом. |
| **Конспект для практики** | `lesson_N/essence_N.md`                 | **Конспект исключительно для практики и отработки**                           |

**Вынос урока** — это подготовка **практического** слоя **`essence_N.md`**.

## 2. Границы урока (диапазон страниц)

- Одна папка **`lesson_N`** соответствует **непрерывному диапазону** страниц учебника.
- Номер файла **`K.png`** = номер **печатной страницы** в книге.
- В шапке **`essence_N.md`** обязательно указывается **диапазон** (например, «стр. 132–149») и название единицы в
  кавычках.

## 3. Назначение `essence_N.md`

**Конспект нужен ТОЛЬКО для практики и отработки/повторения.**

- Это **не** пересказ страниц учебника.
- Это **практическая карта** для активной отработки навыков.
- Это **самодостаточная шпаргалка**, в которой **всё** (структуры, слова, таблицы) уже встроено и готово к использованию
  без учебника.

**Особенно важно для повторных уроков**  
***Повторные уроки (например, 5-я и 10-я единицы) — это именно ещё один полноценный конспект, где всё собрано вместе +
взаимосвязано и взаимопереплетено, чтобы можно было отработать все темы одновременно и «в непривычном антураже»***.

## 4. Структура `essence_N.md` (рекомендуемая)

1. Заголовок `#` с эмодзи + тема + диапазон страниц.
2. Хлебные крошки.
3. Курсивом — 2–4 предложения о том, **что именно можно отработать** в этом конспекте.
4. Таблица «Быстрые ссылки».
5. Раздел **целей** (`## 📌 …`).
6. Дальше только практические блоки:
    - Лексика (структурированные таблицы)
    - Грамматика (таблицы + краткие правила)
    - Структуры и модели предложений
    - Готовые фразы и sentence builders

## 5. Что обязательно вынести в конспект

***Только то, что нужно для практики и отработки:***

- **Структуры** для активного использования (модели предложений, шаблоны, готовые фразы, диалоги).
- **Слова** в структурированном виде: существительные, глаголы, прилагательные, наречия и т.д. (таблицы с примерами в
  контексте). В том числе если это тема урока - в разных видах.
- **Таблицы** грамматики (спряжения, склонения, артикли и т.д.) — переписанные в Markdown + краткое русское правило.
- **Ключевые реплики** и типовые фразы из синих/оранжевых блоков («Πώς το λένε;», «Για δες» и т.п.).
- Для повторных уроков — **полная карта взаимосвязей** + единая таблица всех тем + структуры для комплексной отработки.

**Диалоги целиком НЕ выносим.**  
Нужны только **структуры** и лексика, которые из них можно извлечь. Можно вынести диалог если он *хорошо* отрабатывает
материал. Иначе лучше преструктурировать его для лучшей отработки.

## 6. Чего избегать

- Ссылок на «стр. X, упр. Y» — конспект должен быть полностью самодостаточным.
- Полного копирования диалогов и длинных текстов, если они пустые.
- Пересказа содержания — только практика.

## 7. Связь с `content_N.md`

После добавления `essence_N.md` запускай скрипт:

```bash
python3 scripts/generate_book_lesson_content_md.py
```

- В `content_N.md` появится строка **«💎 Суть урока»** со ссылкой на `essence_N.md`, если файл существует.

## 8. Контрольный список перед коммитом

- [ ] Диапазон страниц в шапке совпадает с `raw/*.png` в папке урока.
- [ ] Конспект читается **без открытия учебника** для основных тренировочных блоков.
- [ ] Таблицы выровнены, ссылки относительные, стиль согласован с [AGENTS.md](../AGENTS.md).
- [ ] Скрипт генерации выполнен, если менялся состав `essence_*.md` или ожидается обновление быстрых ссылок.
