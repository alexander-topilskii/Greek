# 📖 Урок в репозитории (обзор)

*Как устроены слои материала одного урока: подготовка из скана, сборка главы, хаб урока, автономные материалы для читателя, голосовые HTML. Детальные правила вынесены в отдельные файлы.*

*Оглавление каталога `docs`: [README.md](README.md).*

**Детализация:** [essence-generation.md](essence-generation.md) (конспект **`essence_N/essence_N.md`**) · [voice-generation.md](voice-generation.md) (Voice HTML и голосовые сессии) · [voice-lesson-from-digitized.md](voice-lesson-from-digitized.md) (`voice_lesson_N.md`).

---

## 1. Слои материалов

Условно **две фазы**: **подготовка** (сырьё и сборка текста главы) и **этап урока для читателя** (самодостаточные файлы + навигационный хаб).

### 1.1. Подготовка (авторы, пайплайн)

| Слой | Файлы | Назначение |
|------|-------|------------|
| **Скан** | `lesson_N/raw/*.png` | Источник при подготовке: страницы по номеру в колонтитуле. |
| **Постраничная оцифровка** | `lesson_N/digitized/K.md` (опционально) | Текст страницы `K`, формулировки заданий, поиск по уроку. |
| **Глава целиком** | `lesson_N/lesson_digitized/lesson_N_digitized.md` (при наличии) | Единый Markdown главы; вход для генерации **`voice_lesson_N.md`** — см. [voice-lesson-from-digitized.md](voice-lesson-from-digitized.md). |

### 1.2. Этап урока для читателя

| Слой | Файлы | Назначение |
|------|-------|------------|
| **Хаб** | `lesson_N/content_N.md` / `content_N.html` | Список страниц, превью, ссылки на все артефакты урока (сканы, оцифровки, конспект, voice, HTML); генерируется скриптом. |
| **Конспект** | `lesson_N/essence_N/essence_N.md` (опционально) | Практическая замена печатного материала для повторения — по правилам [essence-generation.md](essence-generation.md). **Автономный** файл: без ссылок на сырьё и без перекрёста с `voice_lesson_N.md` внутри markdown. |
| **Голосовой раздаточный материал** | `lesson_N/lesson_voice_N/voice_lesson_N.md` (опционально) | Разговорная отработка по тексту главы — [voice-lesson-from-digitized.md](voice-lesson-from-digitized.md). **Автономный** файл для читателя по тем же правилам «острова». |
| **Voice + HTML** | `lesson_N/essence_N.html`, при наличии `lesson_voice_N/voice_lesson_N.html` (генерируются) | Статические страницы для копирования промпта + текста — см. [voice-generation.md](voice-generation.md). |

**Вынос урока** для авторов — подготовка **`essence_N/essence_N.md`** и при необходимости **`voice_lesson_N.md`** и оцифровок; **сканы** задают границы урока по нумерации страниц.

### 1.3. Место конспекта и голосового урока в цепочке «материал → практика»

| Уровень | Где лежит | Задача |
|---------|-----------|--------|
| Исходник страницы | `raw/K.png`, при необходимости `digitized/K.md` | Точное соответствие скану; полный текст и формулировки со страницы — **в оцифровке**, если нужны для смысла (для авторов репозитория). |
| Конспект | `essence_N/essence_N.md` | Сценарий повторения для читателя — [essence-generation.md](essence-generation.md). Голосовая отработка с конспектом — [voice-generation.md](voice-generation.md). |
| Голосовой урок | `lesson_voice_N/voice_lesson_N.md` | Отдельный автономный материал; навигация к нему — через **`content_N`**. |
| Вне репозитория | Карточки, внешние тренажёры | По желанию; в учебных `.md` урока не дублировать ссылки «пройдите тест на сайте …» как обязательный шаг. |

Конспект и голосовой урок — **не** свод ответов «к номерам в книге» и **не** замена друг другу через перекрёстные ссылки в файлах.

## 2. Границы урока (диапазон страниц)

- Папка **`lesson_N`** = **непрерывный диапазон** страниц, собранный в `raw/`.
- **`K.png`** = номер **печатной** страницы в исходном пособии (имя совпадает с номером в колонтитуле скана); общие
  правила для `raw/` — в [agents.md](../agents.md) (раздел про скриншоты страниц).
- В шапке **`essence_N/essence_N.md`** — **диапазон** (например, «стр. 132–149») и при необходимости **название единицы** в кавычках («Ψάχνω για σπίτι») — подробнее [essence-generation.md](essence-generation.md).
- При смене издания или сомнениях **сверяйте цитаты и диапазон** со сканом в `raw/`; номер в колонтитуле страницы —
  ориентир для имени `K.png`.
