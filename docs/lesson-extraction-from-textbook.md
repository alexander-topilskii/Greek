# 📖 Урок в репозитории (обзор)

*Как устроены слои материала одного урока: скан, оцифровка, навигация, конспект, голосовые HTML. Детальные правила вынесены в отдельные файлы.*

*Оглавление каталога `docs`: [README.md](README.md).*

**Детализация:** [essence-generation.md](essence-generation.md) (конспект `essence_N.md`) · [voice-generation.md](voice-generation.md) (Voice HTML и голосовые сессии).

---

## 1. Слои материалов

| Слой             | Файлы                                    | Назначение |
|------------------|------------------------------------------|------------|
| **Скан**         | `lesson_N/raw/*.png`                     | Источник при подготовке материалов: страницы по номеру в колонтитуле. |
| **Оцифровка**    | `lesson_N/digitized/K.md` (опционально)  | Текст страницы `K`, формулировки заданий из книги, поиск по уроку. |
| **Навигация**    | `lesson_N/content_N.md`                  | Список страниц, превью, ссылки на `raw` и `digitized`; генерируется скриптом. |
| **Суть урока**   | `lesson_N/essence_N.md` (опционально)    | Практическая замена печатного материала для повторения — по правилам [essence-generation.md](essence-generation.md). |
| **Voice + HTML** | `lesson_N/essence_N.html` (генерируется) | Сборка из промпта и конспекта — см. [voice-generation.md](voice-generation.md). |

**Вынос урока** для авторов — подготовка **`essence_N.md`** и при необходимости оцифровки; **сканы** задают границы урока по нумерации страниц.

### 1.1. Место `essence` в цепочке «материал → практика»

| Уровень           | Где лежит                                       | Задача |
|-------------------|-------------------------------------------------|--------|
| Исходник страницы | `raw/K.png`, при необходимости `digitized/K.md` | Точное соответствие скану; полный текст и формулировки заданий со страницы — **в оцифровке**, если нужны для смысла (для авторов репозитория). |
| Конспект          | `essence_N.md`                                  | Сценарий повторения и практическая замена печатного материала для читателя — см. [essence-generation.md](essence-generation.md). Голосовая отработка с конспектом — [voice-generation.md](voice-generation.md). |
| Вне репозитория   | Карточки, внешние тренажёры                     | По желанию; в `essence` не дублировать ссылки «пройдите тест на сайте …» как обязательный шаг. |

Конспект — **опора для повторения вслух и письменно** и автономный учебный текст для читателя, не свод ответов «к номерам в книге».

## 2. Границы урока (диапазон страниц)

- Папка **`lesson_N`** = **непрерывный диапазон** страниц, собранный в `raw/`.
- **`K.png`** = номер **печатной** страницы в исходном пособии (имя совпадает с номером в колонтитуле скана); общие
  правила для `raw/` — в [agents.md](../agents.md) (§9).
- В шапке **`essence_N.md`** — **диапазон** (например, «стр. 132–149») и при необходимости **название единицы** в
  кавычках («Ψάχνω για σπίτι») — подробнее [essence-generation.md](essence-generation.md).
- При смене издания или сомнениях **сверяйте цитаты и диапазон** со сканом в `raw/`; номер в колонтитуле страницы —
  ориентир для имени `K.png`.
