# 📖 Правила выноса «урока» из учебника

**[🏠 Readme](../Readme.md) → [🧭 agents.md](../AGENTS.md) → 📄 `docs/lesson-extraction-from-textbook.md`**

*Методичка для подготовки материалов курса по книге «Ελληνικά Α'»: что считать **единицей урока**, как связаны папки, сканы и конспект, как оформлять **`essence_N.md`**, чтобы им можно было пользоваться **без открытого учебника**.*

| ⚡ Быстрые ссылки | |
|-------------------|---|
| 🧩 Формат `.md` | [AGENTS.md](../AGENTS.md) |
| 📚 Точка входа по уроку | `book/pages/lesson_N/content_N.md` |
| 💎 Конспект | `book/pages/lesson_N/essence_N.md` |
| 🖼 Сканы | `book/pages/lesson_N/raw/{номер}.png` |
| 🔧 Регенерация `content` | [scripts/generate_book_lesson_content_md.py](../scripts/generate_book_lesson_content_md.py) |

## 1. Что такое «урок» в этом репозитории

| Слой | Файлы | Назначение |
|------|--------|------------|
| **Скан** | `lesson_N/raw/*.png` | Источник истины: страницы учебника по номеру в колонтитуле. |
| **Оцифровка** | `lesson_N/digitized/K.md` (опционально) | Дословный или близкий текст со страницы `K` для поиска и цитирования. |
| **Навигация** | `lesson_N/content_N.md` | Список страниц, превью, ссылки на `raw` и `digitized`; генерируется скриптом. |
| **Суть урока** | `lesson_N/essence_N.md` (опционально) | **Конспект для повторения и практики**: тема, грамматика, лексика, готовые фразы и задания **внутри файла**. |

**Вынос урока** в смысле этого документа — прежде всего подготовка **содержательного** слоя (**`essence_N.md`**) и при необходимости **оцифровки**; **сканы** уже задают границы урока по нумерации страниц.

## 2. Границы урока (диапазон страниц)

- Одна папка **`lesson_N`** соответствует **непрерывному диапазону** страниц учебника, собранному в `raw/`.
- Номер файла **`K.png`** = номер **печатной страницы** в книге (см. чеклист переименования в [AGENTS.md §9](../AGENTS.md)).
- В шапке **`essence_N.md`** указывайте **диапазон** (например, «стр. 132–149») и при необходимости **название единицы** из учебника в кавычках («Ψάχνω για σπίτι»).

## 3. Назначение `essence_N.md`

- Это **не** пересказ каждой страницы и **не** замена учебнику для первого прохождения.
- Это **сжатая карта темы**, **шпаргалка**, **таблицы**, **диалоги и упражнения с полным текстом** там, где в книге они только по ссылке на номер упражнения.
- **Самодостаточность:** формулировки вроде «сделайте упражнение 6 на стр. 62» заменяйте на **встроенную цепочку, таблицу или диалог** + при необходимости **ключ** для самопроверки.
- **Повторные уроки** (например, 5-я и 10-я единицы — обзор нескольких блоков) оформляйте как **сводку**: карта «что из какого урока», единая шпаргалка, чеклист навыков.

## 4. Структура `essence_N.md` (рекомендуемая)

Соблюдайте общий стиль репозитория ([AGENTS.md](../AGENTS.md)):

1. Заголовок `#` с эмодзи + краткий подзаголовок (тема, диапазон страниц).
2. **Хлебные крошки** до `Readme` → `book/pages` → `content_N.md` → текущий файл.
3. Курсивом — 2–4 предложения **о чём урок** и для кого конспект.
4. Таблица **«Быстрые ссылки»**: `content_N.md`, диапазон страниц, ссылки на **соседние** `essence_{N±1}.md` или на обзорный урок, если логично.
5. Раздел **целей** (`## 📌 …`).
6. Далее по смыслу: **лексика** (таблицы), **грамматика** (таблицы спряжений/падежей), **фразы**, **встроенные диалоги**, **практика** (с ключом при необходимости), **чеклист**.

Имена разделов можно снабжать эмодзи для навигации, единообразно с другими файлами курса.

## 5. Что обязательно вынести из учебника в конспект

- **Заголовок темы** и **ключевые реплики** из синих/оранжевых блоков («Πώς το λένε;», «Για δες»).
- **Таблицы** грамматики (спряжение, артикли, сравнение степеней и т. д.) — **переписать в Markdown**, не ограничиваться «см. таблицу в книге».
- **Числительные**, **предлоги с артиклями**, **типовые ошибки** — дать **минимум примеров** в предложениях.
- **Диалоги** для разбора — привести **полный текст** в конспекте, если задание в книге дано ссылкой на номер упражнения.

## 6. Чего избегать

- Ссылок **только** на «стр. X, упр. Y» без вставки материала.
- Дублирования **всех** упражнений книги — держите объём разумным; полноту обеспечивают сканы и `content_N.md`.
- Противоречий учебнику: при сомнении в форме глагола ориентируйтесь на **скан** и **оцифровку** данной страницы.

## 7. Связь с `content_N.md`

- После добавления или переименования `essence_N.md` запустите:

  `python3 scripts/generate_book_lesson_content_md.py`

- В `content_N.md` появится строка **«💎 Суть урока»** со ссылкой на `essence_N.md`, если файл существует.

## 8. Контрольный список перед коммитом

- [ ] Диапазон страниц в шапке совпадает с `raw/*.png` в папке урока.
- [ ] Конспект читается **без открытия учебника** для основных тренировочных блоков.
- [ ] Таблицы выровнены, ссылки относительные, стиль согласован с [AGENTS.md](../AGENTS.md).
- [ ] Скрипт генерации выполнен, если менялся состав `essence_*.md` или ожидается обновление быстрых ссылок.
