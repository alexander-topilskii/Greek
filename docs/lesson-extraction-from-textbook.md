## 1. Что такое «урок» в этом репозитории

| Слой | Файлы | Назначение |
|------|--------|------------|
| **Скан** | `lesson_N/raw/*.png` | Источник при подготовке материалов: страницы по номеру в колонтитуле. |
| **Оцифровка** | `lesson_N/digitized/K.md` (опционально) | Текст страницы `K` для поиска и цитирования при выносе. |
| **Навигация** | `lesson_N/content_N.md` | Список страниц, превью, ссылки на `raw` и `digitized`; генерируется скриптом. |
| **Суть урока** | `lesson_N/essence_N.md` (опционально) | **Самодостаточный конспект** для повторения: правила, готовые фразы, словарь; без отсылок читателя к печатному изданию. Объём упражнений с пропусками в курс не копируется целиком — в конспекте держим **готовые формулировки** и при необходимости **сжатые эталоны**. |
| **Voice + HTML** | `lesson_N/essence_N.html` (генерируется) | Не редактировать вручную: собирается из `docs/ai_voice_promt.md` + `essence_N.md` скриптом `scripts/generate_essence_html.py` (вызывается при регенерации `content_N.md`). |

**Вынос урока** для авторов — подготовка **`essence_N.md`** и при необходимости оцифровки; **сканы** задают границы урока по нумерации страниц.

## 2. Границы урока (диапазон страниц)

- Папка **`lesson_N`** = **непрерывный диапазон** страниц, собранный в `raw/`.
- **`K.png`** = номер **печатной** страницы в исходном пособии (см. чеклист в [AGENTS.md §9](../AGENTS.md)).
- В шапке **`essence_N.md`** — **диапазон** (например, «стр. 132–149») и при необходимости **название единицы** в кавычках («Ψάχνω για σπίτι»).

## 3. Назначение `essence_N.md` (для читателя)

- **Самодостаточность:** всё, что нужно для понимания и повторения темы, **внутри файла** (или в явно указанных файлах репозитория — например `digitized/K.md` при полном совпадении с уроком). **Не** писать в конспекте «см. учебник», «откройте книгу».
- Это **не** пересказ каждой страницы подряд и **не** полный сборник решений ко всем упражнениям с пропусками: фокус на **правилах, фразах, лексике** и **коротких эталонах** там, где нужна сверка готовой формы.
- **Практическая карта темы** и опора для повторения вслух и письменно.
- Не отправляйте читателя к «упр. N на стр. M» как к **единственному** источнику смысла: нужный связный текст — **в конспекте** или в `digitized/K.md`.
- **Эталоны:** короткие блоки с готовым текстом (диалог, абзац, таблица выбора) — **без** превращения файла в полный ключ ко всем номерам упражнений блока.
- **Повторные уроки** (обзор) — карта «что из какого номера урока», единая шпаргалка.

## 4. Структура `essence_N.md` (рекомендуемая)

Соблюдайте [AGENTS.md](../AGENTS.md). **Порядок разделов:**

1. Заголовок `#` с эмодзи + тема + диапазон страниц.
2. **Хлебные крошки** до `Readme` → `book/pages` → `content_N.md` → текущий файл.
3. Курсивом — 2–4 предложения **о чём урок** и что файл **самодостаточен** для повторения; **порядок разделов** (цели → … → словарь).
4. Таблица **«Быстрые ссылки»**: `content_N.md`, диапазон, соседние `essence_{N±1}.md` или обзорный урок.
5. **`## 📌 …`** — тема и цели (+ при необходимости компактная сводка целей урока).
6. **`## 🧠 …`** — грамматика и правила.
7. **`## 💬 …`** — готовые фразы и диалоги.
8. **`## 📖 …`** *(опционально)* — эталоны для самопроверки — **сжато**.
9. **`## 📇 …`** — словарь — **последний раздел** файла.

**Мета-текст** (как пояснять устройство файла):

- **Один раз** во вступлении: роль файла и порядок разделов.
- В блоке **эталонов** — одна строка: зачем блок (сверка), без повторения длинных отграничений, если мысль уже вверху.
- **Отсылки «см. выше»** — по названию подраздела или якорю; не дублировать большие таблицы.
- Заголовок **«Словарь»** уже задаёт место лексики; не дублировать это в курсиве в том же заголовке.

**Не закрепляйте** в правилах сценарии внешних голосовых ассистентов.

## 5. Что включить в конспект при выносе из материалов курса

- **Название темы** и **ключевые реплики** из выделенных блоков («Πώς το λένε;», «Για δες»).
- **Таблицы** грамматики в Markdown + краткое правило по-русски.
- **Числительные**, **артикли с предлогами**, типичные ошибки — с **примерами** в предложениях.
- **Диалоги** — полный текст в конспекте, если иначе смысл теряется.

## 6. Чего избегать в тексте `essence_N.md`

- Отсылок к печатному пособию как к **обязательному** источнику для понимания.
- Смысла, который возможен **только** через «упр. N на стр. M» без расшифровки в конспекте или оцифровке.
- Полного дублирования **всех** упражнений с пропусками: это размывает фокус конспекта.
- Противоречий исходным материалам: при сомнении — **скан** и **оцифровка** страницы в репозитории.

## 7. Связь с `content_N.md`

- После добавления или переименования `essence_N.md` запустите:

  `python3 scripts/generate_book_lesson_content_md.py`

- В `content_N.md` появятся строки **«💎 Суть урока»** (на `essence_N.md`) и **«🎙 Voice (HTML)»** (на `essence_N.html` и общий индекс), если файл конспекта существует. Обновляется также `essence_N.html` и [essence_voice_index.html](../book/pages/essence_voice_index.html).

## 8. Контрольный список перед коммитом

- [ ] Диапазон страниц в шапке совпадает с `raw/*.png` в папке урока.
- [ ] Конспект **самодостаточен** для повторения основных блоков темы.
- [ ] Таблицы выровнены, ссылки относительные, стиль согласован с [AGENTS.md](../AGENTS.md).
- [ ] Скрипт генерации выполнен при изменении `essence_*.md` (пересобраны `content_*.md`, `essence_*.html`, индекс Voice).
